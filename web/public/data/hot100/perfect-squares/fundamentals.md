# 📚 基础百科：完全平方数小词典

### 🌳 核心概念比喻：拼图规则

在我们的“拼图之旅”中，记录仪会帮我们记下这件事：
*   **积木形状 (Square Numbers)**：只能用 $1\times1, 2\times2, 3\times3$ 等平方大小的积木。
*   **拼图策略 (Minimization)**：为了拼出同样的高度，积木用得越少说明拼得越妙！

---

### 🔍 变量小词典：
| 变量名 | 它的“外号” | 它是做什么的？ |
| :--- | :--- | :--- |
| `n` | 目标塔高 | 我们最后要拼出的总高度。 |
| `dp` | 省力记账本 | 一个列表，记录拼到每一级高度时最少需要的积木数。 |
| `square` | 积木箱子 | 提前准备好的一堆不同大小的完全平方数积木（1, 4, 9...）。 |
| `dp[i]` | 第 i 级方案 | 拼出高度为 i 时最省力的方案。 |

---

### 🧙 魔法公式：
*   **初始记忆**：`dp[0]=0`（高度为 0 不需要积木）。
*   **减法魔法**：`dp[i] = min(dp[i], dp[i-s] + 1)`。
    *   `s` 是我们要试放的一块积木。
    *   `dp[i-s]` 是放这块之前的最优方案。
    *   `+ 1` 是算上当前放下的这块积木。
*   **寻找最小值**：我们要试过箱子里所有的 `s`，挑出一个让 `dp[i]` 最小的结果。
