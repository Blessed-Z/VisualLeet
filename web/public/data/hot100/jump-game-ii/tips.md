# 💡 思路巧记：极速跳跃找寻诀

### 🎼 朗朗上口小口诀
> **“走在边界里，**
> **心中有未来。**
> **走到边界时，**
> **跳一跳，最快开！”**

---

### 🔑 核心关键点：
1. **贪心区间**：关注“当前一步”能到达的最大范围。
2. **两级边界**：
    *   `cur_right`：当前这一步能跳到的最远位置。
    *   `nex_right`：在当前步的范围内，如果再跳一步能去到的最远位置。
3. **跳跃时机**：遍历数组，当走到 `cur_right`（不得不跳了）时，增加跳数 `ans += 1`，并更新 `cur_right = nex_right`。
4. **提前终止**：遍历范围到 `len(nums) - 1` 即可。因为如果已经站在终点，就不需要再跳一次了。
5. **最少次数**：这就是典型的贪心，每一步都在为下一步争取最大的“生存空间”。
