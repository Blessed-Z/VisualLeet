# 148. 排序链表：给小朋友排排队 (归并排序)

想象一下，你面前站着一队乱糟糟的小朋友，他们身高各异，我们需要把他们按照从矮到高的顺序排好。

### 🏫 故事背景：分而治之
如果我们直接在大队里换来换去，肯定会乱套。这时候，我们可以用一个**“分而治之”**的绝招：

1.  **分小队**：先把这队小朋友分成左、右两半。如果每半还是很多人，就继续分，直到每个小队只有**1个小朋友**。
2.  **小比赛**：1个小朋友当然是“自带有序”的。
3.  **大合并**：把两个有序的小队合并成一个更大的有序队。就像两个已经排好队的小组进场，我们只需要比较两个队头谁更矮，谁就先站到新队伍里。

### 🪄 魔法步骤

1.  **找中点 (快慢指针)**：
    *   派出两个侦察兵：**小慢**（一次走一步）和**小快**（一次走两步）。
    *   当小快走到终点时，小慢刚好在队伍中间！
2.  **切断连接**：
    *   在中点把链表剪断，变成两个独立的短链表。
3.  **递归魔法**：
    *   对自己左边和右边的链表分别念动“排序咒语”。
4.  **丝滑合并**：
    *   用“合并两个有序链表”的技巧，把两段排好序的链表缝合在一起。

### 💡 为什么这么做？
归并排序是处理链表最优雅的方式，因为它不需要像数组那样频繁移动元素，只需要改变指针的指向，就像小朋友们互相牵手一样简单！
