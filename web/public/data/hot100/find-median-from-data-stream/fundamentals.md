# 📚 基础百科：中位数流小词典

### 🌳 核心概念比喻：两堆平衡球

在我们的“数据流”中，就像是一个正在不停掉落小球的管道：
*   **大顶堆 (Max Heap)**：就像是一个魔法瓶子，它把所有**较小**的球装在一起。
    *   这个瓶子的盖子在最上面，也就是**最大**的那个球。
    *   它的作用是守住“下半场”的边界。
*   **小顶堆 (Min Heap)**：就像是另一个魔法瓶子，它把所有**较大**的球装在一起。
    *   这个瓶子的盖子也在最上面，也就是**最小**的那个球。
    *   它的作用是守住“上半场”的边界。

### 🔍 变量小词典：
| 变量名 | 它的“外号” | 它是做什么的？ |
| :--- | :--- | :--- |
| `self.left` | 矮人国 (大顶堆) | 存入较小的一半数字（Python 中存负值）。 |
| `self.right` | 巨人国 (小顶堆) | 存入较大的一半数字。 |
| `heappush` | 进场 | 把新朋友扔进对应的魔法瓶子里。 |
| `heappushpop` | 过水 | 先把一个数扔进瓶子，再把瓶盖那个数拿出来。 |
| `findMedian` | 裁判看中间 | 看看两个瓶盖上的数，算出中位数。 |

---

### 🧙 魔法公式：
*   **左边人多**：中位数 = `-self.left[0]`。
*   **一样多**：中位数 = `(-self.left[0] + self.right[0]) / 2.0`。
*   **注意**：Python 的 `heapq` 默认只有小顶堆。为了变成大顶堆，我们要存入 `-num`。这样原来的最大值就会变成最小的负数，呆在堆顶啦！
