# 📚 基础百科：极速跳跃小词典

### 🌳 核心概念比喻：两级边界

在我们的“跳跃之旅”中，记录仪会帮我们记下两级边界：
*   **当前边界 (Current Boundary)**：你现在这一次跳跃，最远能落在哪里。
*   **下一级预测 (Next Potential Boundary)**：在你现在的活动范围内，如果再跳一次，最远能去哪里。

---

### 🔍 变量小词典：
| 变量名 | 它的“外号” | 它是做什么的？ |
| :--- | :--- | :--- |
| `nums` | 砖块跳跃力 | 每块砖头能给你提供的下一跳力量。 |
| `cur_right` | 当前极限 | 你正在跳的这一步，最远能停在哪。 |
| `nex_right` | 未来极限 | 如果你从当前范围内挑一块最好的砖头，下一步能去哪。 |
| `ans` | 跳跃次数 | 你一共跳了几次。 |

---

### 🧙 魔法公式：
*   **预测未来**：`nex_right = max(nex_right, nums[i] + i)`（在每一块砖上，都通过望远镜预测下一跳的最远距离）。
*   **不得不跳了**：`if i == cur_right:`（当你走到当前跳跃的最边缘了）。
*   **更新状态**：`ans += 1; cur_right = nex_right`（跳一下，开启下一段更远的旅程）。
