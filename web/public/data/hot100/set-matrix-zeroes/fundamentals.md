# 📖 基础百科：矩阵的逻辑坐标

## 💡 什么是矩阵？
矩阵就像是一个**围棋盘**或者**Excel表格**。
- `matrix[i][j]`：代表第 `i` 行、第 `j` 列的那个位置。
- **行（Row）**：横向的，也就是 $i$。
- **列（Column）**：纵向的，也就是 $j$。

## 📋 变量小字典
- **`m, n`**：大楼的高和宽（行数和列数）。
- **`flag_row / flag_col`**：两个“紧急备忘录”。
- **`matrix[i][0]`**：第 $i$ 层的门口公告栏。
- **`matrix[0][j]`**：第 $j$ 列的楼顶公告栏。

---

## 📈 进阶思考
为什么要从索引 1 开始遍历？
- 因为索引 0（第一行和第一列）被我们拿来当“公告栏”了！如果从 0 开始，公告栏的信息就会和房间的信息混在一起，导致逻辑混乱。这就是算法中的“空间复用”技巧。
