# 2. 两数相加 (The Math Machine)

## 💡 深度解析
小朋友，想象你有两个存钱罐，里面的硬币是按个位、十位、百位...逆序排好的。
我们要把这两个存钱罐合并，算出一个总数。

### 核心策略：竖式加法
这就像你在草稿纸上算加法：
1. **从个位开始**：把两个存钱罐里最前面的硬币拿出来加一下。
2. **处理进位**：如果加起来超过了 9，你就得拿出一个小本子记下“进位 1”，然后把剩下的余数存进新罐子。
3. **依次往后**：每一位都要加上刚才小本子里的进位。
4. **最后一眼**：如果你走到了尽头，手里还有一个进位没处放，记得给它单独做一个新格子！

---

## 🛠️ 执行步骤
1. 创建一个 `dummy` 节点作为结果的开头。
2. 设定 `carry = 0` 记录进位。
3. 同时遍历两个链表：
   - 没走完的取值，走完的取 0。
   - `新值 = (a + b + carry) % 10`。
   - `新进位 = (a + b + carry) // 10`。
4. 创建新节点并挂载。
5. 返回结果链表。

---

## 📈 复杂度分析
- **时间复杂度**：$O(max(N, M))$。取决于两个数字里更长的那个。
- **空间复杂度**：$O(1)$。除了存储结果的链表，我们只用了几个变量。
