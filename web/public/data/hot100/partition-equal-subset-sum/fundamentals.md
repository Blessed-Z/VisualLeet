# 🧩 分割等和子集基础百科

### 🌟 核心概念：什么是 0/1 背包问题？

想象你是一个小盗贼，面前有一堆宝物。每样宝物你只能选一次（要么拿，要么不拿，这就是 0 和 1）。
在这道题里，“重量”就是数字本身，“背包容量”就是总和的一半。
我们要看这个背包能不能被宝物**刚好填满**。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `total` | **总重量** | 数组中所有数字的和。 |
| `target` | **平衡点** | 每一组必须达到的重量 (`total // 2`)。 |
| `dp` | **重量记录本** | 一个布尔数组，记录某个重量能否被凑出来。 |
| `bits` | **位集魔法棒** | 一个整数，利用它的二进制位来代表重量状态。 |
| `x` | **当前拿到的金币** | 数组中的每一个数字。 |

---

### ⚠️ 敲黑板

*   **正整数**：因为金币重量必须是正数。
*   **提前退出**：如果某个金币本身就比 `target` 还大，那肯定分不平。
*   **位运算公式**：`bits |= (bits << x)`。
    *   `bits << x` 相当于：把目前所有能凑出的重量，都加上一个新的 `x`。
    *   `|=` 相当于：把原来的状态和新增的状态合在一起。
