# 📚 构造二叉树基础百科

### 🌟 核心概念：遍历顺序
*   **前序 (Preorder)**: 根 -> 左 -> 右。所以根节点总是排在最前面。
*   **中序 (Inorder)**: 左 -> 根 -> 右。所以根节点把左右子树完美地隔开了。

---

### 📖 变量小字典：

*   **`preorder`**: 节点的出生顺序名单。
*   **`inorder`**: 节点的座位排次名单。
*   **`k`**: 根节点在中序名单里的索引，也是左子树的小朋友总数。
*   **`creatTree`**: 我们的建筑师函数，不断地拆分名单并盖楼。

---

### 🔍 递归的边界：
当名单变为空（长度为 0）时，说明这一边已经没有小朋友了，建筑师就可以收工返回 `None`。
