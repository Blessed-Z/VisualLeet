# 💡 寻找重复数巧记

### 🎵 押韵口诀
> 重复数字藏得深，数组变身链表轮。
> 乌龟一步兔两步，环中相遇定乾坤。
> 乌龟回巢兔原地，同步向前再找门。
> 二次碰头那一刻，重复数字现真身。

---

### 🔑 核心关键点
1.  **映射关系**：将 `i -> nums[i]` 看作是节点 $i$ 指向节点 $nums[i]$ 的链表。
2.  **必然有环**：因为有 $n+1$ 个数且都在 $1$ 到 $n$ 之间，根据鸽巢原理，一定有重复，即一定有环。
3.  **不能修改数组**：题目限制了不能排序、不能修改原数组（如取反法），因此快慢指针是最优解。
4.  **起点选择**：通常从下标 0 开始，即 `slow = nums[0]`, `fast = nums[nums[0]]`。
