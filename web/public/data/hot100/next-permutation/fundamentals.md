# 🧩 下一个排列基础百科

### 🌟 核心概念：什么是字典序？

字典序（Lexicographical Order）就是像查字典一样的排序方法。
对于数字排列，字典序小的排在前面（如 `123 < 132 < 213`）。
“下一个排列”就是要找到在所有可能的排列中，刚好比当前排列大的那一个。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `nums` | **数字卡片** | 输入的待调整数组。 |
| `i` | **转折点 (下降点)** | 从右向左第一个数值变小的位置。 |
| `j` | **备胎 (大数)** | 在 `i` 右侧，刚好比 `nums[i]` 大一点的数。 |
| `left/right` | **翻转边界** | 用于原地反转数组部分的双指针。 |

---

### ⚠️ 敲黑板

*   **局部最优 vs 全局最优**：我们要让数字变大的幅度最小，所以交换的位置要尽可能靠右。
*   **单调性**：在 `i` 被选定后，`i` 右侧的数字一定是单调递减的。这就是为什么反转它们就能得到最小排列。
*   **原地算法**：题目要求空间复杂度 $O(1)$，不能复制数组。
