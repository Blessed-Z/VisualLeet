# 📚 基础百科：划分小词典

### 🌳 核心概念比喻：家族守护线

在我们的“剪草之旅”中，记录仪会帮我们记下两级边界：
*   **最后一次面孔 (Last Appearance)**：在这个草坪上，每个字母最后出现在哪个位置。
*   **当前家族边界 (Current Partition Boundary)**：为了把目前这一段里所有人的家人都包进来，最远我们要剪到哪里。

---

### 🔍 变量小词典：
| 变量名 | 它的“外号” | 它是做什么的？ |
| :--- | :--- | :--- |
| `s` | 彩色草坪 | 由不同字母组成的彩色草坪。 |
| `last` | 家族名录 | 哈希表，记录每个字母最后出现的格数。 |
| `start` | 切割起点 | 当前这一段草坪从第几格开始。 |
| `end` | 当前终点 | 为了包住全家人，目前这一段至少要剪到哪。 |
| `ans` | 剪草清单 | 每剪一刀，就把这一段的长度记在这里。 |

---

### 🧙 魔法公式：
*   **发现新家属**：`end = max(end, last[c])`（为了包住新遇到的这个字母，我们的终点可能得再往后延）。
*   **全家到齐**：`if i == end:`（当我们的脚步追上了目前定下的终点）。
*   **咔嚓一剪**：`ans.append(end - start + 1); start = i + 1`（剪下一段，并准备开始新的一段）。
