# 📚 基础百科：股票小词典

### 🌳 核心概念比喻：寻宝记录仪

在我们的“寻宝之旅”中，记录仪会帮我们记下两件事：
*   **最低谷 (Valley)**：到目前为止，我们遇到过的最便宜的宝石。
*   **最大巅峰 (Peak)**：如果在这个最便宜的时候买，在今天卖，能赚到的最多钱。

---

### 🔍 变量小词典：
| 变量名 | 它的“外号” | 它是做什么的？ |
| :--- | :--- | :--- |
| `prices` | 每天的价格 | 一个存有宝石价格的列表。 |
| `min_price` | 历史最便宜 | 你在这趟旅程中见过的最低价。 |
| `max_profit` | 最赚的那笔钱 | 到目前为止，最高的利润记录。 |
| `price` | 今天的价钱 | 正在观察的宝石的价格。 |

---

### 🧙 魔法公式：
*   **如果今天更便宜**：`if price < min_price: min_price = price`（把这个最便宜的价格记下来）。
*   **如果卖掉更赚钱**：`elif price - min_price > max_profit: max_profit = price - min_price`（看看能不能破纪录）。
*   **最终答案**：就是走完全程后的那个 `max_profit`。
