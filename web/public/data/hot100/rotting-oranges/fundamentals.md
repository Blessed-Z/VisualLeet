# 📚 广度优先搜索基础百科

### 🌟 核心概念：广度优先搜索 (BFS)
BFS 就像是**丢进水里的石头产生的波纹**。第一分钟扩散一圈，第二分钟再往外扩散一圈。它总是先探索离自己最近的，然后再探索稍微远一点的。
这在解决“最短时间”、“最少步数”的问题时是无敌的！

---

### 📖 变量小字典：

*   **`rotten`**: 僵尸大本营（集合）。里面存着当前这一分钟可以出去咬人的僵尸坐标。
*   **`flesh` (fresh)**: 幸存者营地（集合）。里面存着所有还没被咬的新鲜橘子坐标。
*   **`time`**: 秒表，记录经过了多少分钟。
*   **`[(0,1),(1,0),(-1,0),(0,-1)]`**: 僵尸的四颗牙齿（上下左右四个移动方向）。

---

### 🔍 集合 (Set) 的妙用：
代码中用了 `flesh -= rotten`。这是 Python 集合的魔法，叫**差集**。意思是：“把幸存者营地里，那些刚刚变成僵尸的人，瞬间踢出去！”
