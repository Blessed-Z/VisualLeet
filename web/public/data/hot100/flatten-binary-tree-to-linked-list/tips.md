# 🍡 展开为链表巧计：反向遍历口诀

### 🎼 朗朗上口口诀
> **二叉变链表，倒序穿糖果。**
> **右左根相见，prev 把手牵。**
> **左边全切掉，右边一线连。**

---

### 🔑 核心三招
1. **反向前序遍历**：不是“根-左-右”，而是它的倒序“右-左-根”。
2. **全局变量 `prev`**：用来记录上一次处理好的节点，它就是当前节点的“下一个” (next)。
3. **断开左子树**：处理每个节点时，必须设置 `node.left = None`，否则它还是个树结构。

---

### 🚀 避坑小贴士
*   **不要返回新树**：题目要求原地 (in-place) 修改，不用创建新节点。
*   **右孩子先走**：如果不先递归处理右子树，等你处理完根节点的 left 指向，原来的 right 就找不到了！
*   **递归深度**：如果树很高，注意 Python 的递归限制。但在 LeetCode 中这通常没问题。
