# 98. 验证二叉搜索树 🔍

### 10 岁的小朋友，你好呀！今天我们来当“数字森林”的治安小警察！

在一个完美的“二叉搜索树”森林里，每个节点都要遵守严格的家规。如果不遵守，森林就会乱套！

### 🌲 什么是“森林家规”？
1.  **左边的孩子**：必须比爸爸小。
2.  **右边的孩子**：必须比爸爸大。
3.  **爷爷的叮嘱**：不仅仅是爸爸，所有的祖先都有要求。比如，你是爷爷的左子树里的节点，那你全家都得比爷爷小！

### 🧙‍♂️ 魔法光环 (Magic Halo)
为了检查大家有没有违规，我们要给每个节点带上一个“魔法光环”。这个光环规定了一个**安全范围** `(min, max)`。

-   **根节点**：他是森林的创始者，范围最大，是 `(-无穷, +无穷)`。
-   **去左边分队**：如果你往左走，你的**最大值**就要更新为爸爸的值（因为左边都要比爸爸小）。
-   **去右边分队**：如果你往右走，你的**最小值**就要更新为爸爸的值（因为右边都要比爸爸大）。

### 🚀 检查步骤：
1.  带上你的魔法光环进入节点。
2.  检查你的值是否在 `(min, max)` 之间。如果不在，**哔哔！违规！**
3.  带着更新后的光环去检查左孩子。
4.  带着更新后的光环去检查右孩子。
5.  如果全家人都合格，这棵树就是完美的！

这就像每个人出生时都有一个活动范围，如果你跑出了这个范围，就说明你走错家门啦！
