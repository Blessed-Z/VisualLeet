# 🎭 最长有效括号：舞伴连连看！

想象我们在举办一场跳舞比赛，只有两种角色：
*   `(`：**等待中的舞者**（他在等一个搭档）。
*   `)`：**入场的搭档**（他来找人跳舞）。

我们要找的是：**最长的一排**成功配对且连续的舞者。

---

### 💡 魔法思路：记忆栈

我们准备一个**魔法桶 (栈)**，里面不放括号，而是放他们的**位置编号 (索引)**。

1.  **地基 (-1)**：
    *   我们在桶底先放一个 `-1`。这就像是一个“起跑线”，告诉我们现在的有效括号是从哪里开始计算的。

2.  **遇到 `(` (等待者)**：
    *   非常简单，把他的位置编号丢进桶里，让他排队。

3.  **遇到 `)` (搭档来了)**：
    *   **先从桶里拿走一个编号**（这代表尝试配对）。
    *   **桶空了吗？**
        *   **空了**：说明这个 `)` 是多出来的，没人跟他跳舞。他成了一个新的“地基”，把他的位置编号放进桶底。
        *   **没空**：说明配对成功啦！现在桶顶的那个编号，就是这串“舞者”的**前一个分界点**。
        *   **算长度**：当前位置 - 桶顶编号 = 这一串成功跳舞的人数。

---

### 🎨 举个例子：`()(())`

1.  **准备**：栈 = `[-1]`
2.  **`(` (位置0)**：入栈。栈 = `[-1, 0]`
3.  **`)` (位置1)**：出栈 (0)，栈 = `[-1]`。长度 = `1 - (-1) = 2`。
4.  **`(` (位置2)**：入栈。栈 = `[-1, 2]`
5.  **`(` (位置3)**：入栈。栈 = `[-1, 2, 3]`
6.  **`)` (位置4)**：出栈 (3)，栈 = `[-1, 2]`。长度 = `4 - 2 = 2`。
7.  **`)` (位置5)**：出栈 (2)，栈 = `[-1]`。长度 = `5 - (-1) = 6`。

最大长度就是 6！一气呵成！🎉
