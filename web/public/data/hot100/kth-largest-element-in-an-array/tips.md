# 💡 思路巧记：第 K 大王找寻诀

### 🎼 朗朗上口小口诀
> **“大顶堆，山顶王，**
> **一次建堆最重要。**
> **k-1 次把顶让，**
> **最后堆顶就是宝！”**

---

### 🔑 核心关键点：
1. **原地建堆**：不用开辟新数组，利用索引关系直接把原数组变成大顶堆！
2. **堆化（Heapify）**：从最后一个非叶子节点（`n//2-1`）开始，向上调整，让大数慢慢“浮”上去。
3. **交换与缩小**：每次把堆顶（最大的数）放到数组最后，然后缩小堆的范围，重新调整堆顶。
4. **节省功夫**：只需要把最大的前 k-1 个数排到后面，第 k 个大数自然就在堆顶等着你啦！
