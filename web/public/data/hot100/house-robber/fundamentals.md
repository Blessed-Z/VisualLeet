# 📚 基础百科：打家劫舍小词典

### 🌳 核心概念比喻：怪兽的行囊

在我们的“寻宝之旅”中，记录仪会帮我们记下这件事：
*   **相邻房警戒线 (Non-adjacent Constraint)**：规定不能连续拿两间相邻房子的钱。
*   **行囊价值 (Max Wealth)**：到目前为止，在遵守规矩的前提下，能装下的最多金币。

---

### 🔍 变量小词典：
| 变量名 | 它的“外号” | 它是做什么的？ |
| :--- | :--- | :--- |
| `nums` | 房子的金币 | 每一间房子里放了多少金币。 |
| `dp` | 决策记账本 | 一个列表，记录走到每一间房时，能拿到的最高金额。 |
| `dp[i]` | 走到第 i 间 | 目前最优的拿法攒了多少钱。 |

---

### 🧙 魔法公式：
*   **初始记忆**：`dp[0]=0, dp[1]=nums[0]`（第一间房只有一个选择）。
*   **二选一魔法**：`dp[i] = max(dp[i-1], dp[i-2] + nums[i-1])`（要在“不拿这间拿前一间”和“拿这间加前前一间”中做个选择）。
*   **最终结果**：就是走完全程后，看看记账本最后一页 `dp[size]`。
