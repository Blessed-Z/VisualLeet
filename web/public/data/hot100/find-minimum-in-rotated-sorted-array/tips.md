# 💡 寻找旋转最小值巧记口诀

### 🌟 核心技巧：
> **“找最小，看右边。中值大于右边界，说明断崖在右边，左手跨过中值点；中值小于右边界，说明右路很安全，右手缩回中值处；最后碰头那一刻，谷底数值现真颜。”**

### 🧠 记忆关键点：
1. **右边界基准**：始终让 `nums[mid]` 和 `nums[right]` 比高低。
2. **左闭右闭 vs 左闭右开**：本题使用 `while left < right`，这样最后 `left == right` 直接就是答案。
3. **步长差异**：`left = mid + 1` 而 `right = mid`。因为 `mid` 可能是最小值，所以收缩右边界时不能跳过它。
