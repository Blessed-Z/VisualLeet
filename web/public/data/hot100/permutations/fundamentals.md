# 📚 回溯法基础百科

### 🌟 核心概念：回溯 (Backtracking)
回溯法就像一种**全自动的寻宝机器人**。它会尝试每一条路：
1.  如果这条路能通向宝藏（满足条件），它就把宝藏记下来。
2.  如果这条路是死胡同（不满足条件或超出范围），它就**退回到上一个路口**，假装没走过这条路，然后换一条路继续试。

---

### 📖 变量小字典：

*   **`res`**: 我们的宝藏大箱子，存下所有找到的答案。
*   **`path`**: 机器人的探险路径，记录当前走了哪些步。
*   **`backtracking(...)`**: 机器人的大脑逻辑，决定怎么走、什么时候退。

---

### 🔍 为什么需要 `path[:]`？
在 Python 里，列表是引用传递。如果你直接 `res.append(path)`，箱子里存的其实是同一个篮子的“位置”。当机器人退回原点把篮子倒空时，箱子里之前存的东西也会跟着变空。所以我们需要**拍一张照片（切片副本 `path[:]`）**存进箱子。
