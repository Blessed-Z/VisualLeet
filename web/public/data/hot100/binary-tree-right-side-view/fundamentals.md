# 🌲 二叉树的基础百科

### 🧙 变量小字典 (The Magic Map)
| 变量名 | 它代表什么？ | 比喻 |
| :--- | :--- | :--- |
| `root` | 树的根节点 | 树的最顶端。 |
| `queue` | 队列 | 一排排站好的节点，先进先出。 |
| `result` | 最终结果 | 我们的观察笔记，记录下看到的每一个糖果。 |
| `level_size` | 当前层的节点数量 | 像每一层有几个小房间一样。 |
| `node` | 当前正在观察的节点 | 正在检查的小糖果。 |

---

### 📖 核心概念：层序遍历 (BFS)
层序遍历就像是**坐电梯一层一层下楼**。在每一层楼，电梯门打开，你会看到所有的住户。

- **广度优先搜索 (BFS)**：先看完这一层，再去看下一层。
- **层序遍历特点**：能够清晰地知道每一层都有哪些节点。

---

### 🔍 为什么看右边？
在一个二叉树中，每一层最右边的节点，就是当你从右侧看向这棵树时唯一不被遮挡的那个。
通过层序遍历，我们可以在遍历到每一层的最后一个节点时，顺手把它“抓”出来，放进结果列表里。
