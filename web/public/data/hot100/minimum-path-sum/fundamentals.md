# 🧩 最小路径和基础百科

### 🌟 核心概念：什么是路径权重？

在网格中，如果每个格子都有一个值，我们要找一条路径使得这些值的总和最小，这就是“路径权重”最小化。
这道题是经典的动态规划问题，通过把大的路线拆解成每一小步的最优选择来实现。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `grid` | **金币迷宫** | 输入的格子数组，里面的数字代表每个格子的“重量”。 |
| `dp` | **消耗记录单** | 一个二维数组，用来存走到每一个格子的最小总消耗。 |
| `dp[i][j]` | **当前格最低消耗** | 走到坐标 (i, j) 最少需要收集的金币数。 |
| `grid[i][j]` | **当前格路费** | 经过当前格子必须交的金币数。 |
| `min(dp[i-1][j], dp[i][j-1])` | **最划算来源** | 比较上面和左边，看哪个之前累积的金币更少。 |

---

### ⚠️ 敲黑板

*   **非负整数**：这意味着你走得越多，金币只会越多（或者不变），不会减少。
*   **边界条件**：千万别忘了处理第一行和第一列，它们可是路径的唯一入口。
*   **空间优化**：也可以优化成一维数组，因为计算 `dp[i][j]` 只需要上一行和当前行的左侧数据。
