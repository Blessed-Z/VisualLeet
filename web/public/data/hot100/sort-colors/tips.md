# 💡 颜色分类巧记

### 🎵 押韵口诀
> 荷兰国旗三色染，三指针法是重点。
> 左边红 0 右边蓝，白 1 稳坐正中间。
> 换到左边向前走，换到右边再看眼。
> 指针相遇任务完，原地排序不费钱。

---

### 🔑 核心关键点
1.  **三指针**：`p0` (指向 0 的右边界)，`p2` (指向 2 的左边界)，`curr` (当前遍历位置)。
2.  **原地修改**：题目要求空间复杂度为 $O(1)$，不能使用额外的计数数组（虽然计数排序也能过，但不符合“进阶”要求）。
3.  **注意点**：与 `p2` 交换后，`curr` 指针**不能递增**，因为交换回来的数可能是 0 或 2，需要重新判断。
4.  **循环条件**：`curr <= p2`。
