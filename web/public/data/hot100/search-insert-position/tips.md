# 💡 搜索插入位置巧记口诀

### 🌟 核心技巧：
> **“左闭右开定边界，对折寻找不费力；比它小就左加一，不比它小右靠拢；最后左手定江山，插队位置在这里。”**

### 🧠 记忆关键点：
1. **边界选择**：本题推荐使用 `left, right = 0, len(nums)` 的左闭右开写法，因为 `target` 有可能插入在数组的最末尾（索引为 `len(nums)`）。
2. **比较逻辑**：为了找插入位置，其实我们是在找“第一个大于等于 target 的数”的索引。
3. **循环条件**：`while left < right`，当循环结束时，`left == right`，这就是答案。
