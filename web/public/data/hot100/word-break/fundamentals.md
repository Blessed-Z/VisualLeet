# 🧩 单词拆分基础百科

### 🌟 核心概念：什么是动态规划 (DP)？

动态规划就像是你在解一道超级超级大的题，但是你把它**拆成了很多个以前解过的小题**。
在这道题里，我们要知道“这一整串蛋糕能不能拼成”，最好的办法就是先知道“前半截能不能拼成”，然后再看“后半截模具有没有”。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `s` | **长蛋糕** | 需要被拆分的整串字母。 |
| `wordDict` | **模具箱** | 我们能用的所有合法单词。 |
| `dp` | **拼图记录单** | 一个布尔值数组（True 或 False），记录每一个位置能不能成功拼出来。 |
| `dp[i]` | **小标记** | 表示字符串的前 `i` 个字符是否能由字典里的单词拼成。 |
| `i` | **扫描仪起点** | 尝试从这个位置开始找模具。 |
| `j` | **扫描仪终点** | 尝试切蛋糕的位置。 |

---

### ⚠️ 敲黑板

*   **字符串切片**：在 Python 中，`s[i:j]` 是不包含 `j` 那个位置的字母的哦！
*   **Set 提速**：列表 (List) 找东西要一个个比对，集合 (Set) 找东西像刷身份证，快得飞起！
