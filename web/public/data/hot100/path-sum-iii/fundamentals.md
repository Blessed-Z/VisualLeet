# 📚 路径总和基础百科

### 🌟 核心概念：前缀和 (Prefix Sum)
前缀和就像你跑步时的**计时器**。虽然你只想要某一段路的时间，但如果你记下了从起点到每一个路口的时间，用两个路口的时间相减，就能算出中间那一段花的时间啦！

---

### 📖 变量小字典：

*   **`ans`**: 我们的宝藏箱，记录一共找到了多少条达标路径。
*   **`cnt`**: 神奇账本（哈希表/字典），记录每个“总金额”出现的次数。
*   **`s`**: 你现在兜里所有的金币总数。
*   **`targetSum`**: 我们的目标，希望找到和为这个数的路径。

---

### 🔍 为什么 `cnt[0]=1`？
如果你现在的总金额刚好就是 `targetSum`，那么 `s - targetSum` 就是 0。如果账本里没有 0 的记录，这段从头开始的路径就会被漏掉。所以我们要预设 0 出现了一次。
