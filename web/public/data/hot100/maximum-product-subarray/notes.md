# 🎢 乘积最大子数组：财富过山车！

想象你在坐一个**财富过山车**。轨道上有很多数字，每经过一个数字，你的财富就会乘以这个数字。

*   **正数**：是助推器，让你的财富翻倍。
*   **负数**：是**“镜像传送门”**！它会把大笔财富瞬间变成巨额债务，但也会把巨额债务瞬间变成惊人财富！
*   **零**：是黑洞，会让一切归零。

因为负数的存在，我们不能只盯着现在的最高点，还得盯着现在的**最低点**，万一遇到下一个负数，最低点就能反杀变成最高点！

---

### 💡 魔法思路：双保险动态规划

我们要请两个保镖来保护我们的财富：

1.  **大白 (max_num)**：时刻记录到目前为止，能达到的**最高财富**。
2.  **小黑 (min_num)**：时刻记录到目前为止，跌到的**最惨谷底**（最大的负数）。

**每走一步（遇到数字 x）：**
*   如果 **x 是正数**：大白继续变大，小黑继续变小。
*   如果 **x 是负数**：大白和小黑**交换身份**！大白乘以负数变小，小黑乘以负数反而变大。
*   别忘了，你也可以**从头开始**（如果当前的 x 本身就比乘以之前的财富更给力）。

---

### 🎨 财富翻转瞬间

数字序列：`[2, 3, -2, -4]`

1.  **遇到 2**：大白=2，小黑=2。
2.  **遇到 3**：大白=6，小黑=3（或者说小黑=2*3=6，取最小也是6）。
3.  **遇到 -2**：
    *   大白想：`6 * -2 = -12`，或者自己重来 `-2`。所以大白暂时是 `-2`。
    *   小黑想：`6 * -2 = -12`，或者自己重来 `-2`。所以小黑暂时是 `-12`。
4.  **遇到 -4**：
    *   大白看小黑：`-12 * -4 = 48`！哇，小黑立功了！大白变 `48`。
    *   小黑看大白：`-2 * -4 = 8`，或者自己重来 `-4`。小黑变 `-4`。

最后大白达到的最高记录就是 `48`！🏆
