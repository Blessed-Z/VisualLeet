# 📚 寻找最小值基础百科

### 🌟 核心概念：寻找跳变点
在旋转数组中，最小元素其实就是那个“跳变点”——它是整个数组中唯一一个比左边邻居还要小的数字。

---

### 📖 变量小字典：

*   **`left`**: 搜寻区的左界。
*   **`right`**: 搜寻区的右界。
*   **`mid`**: 黄金分割点。
*   **`nums[right]`**: 我们的“标杆”。只要 `mid` 比标杆大，最小值就一定在右边。

---

### 🔍 为什么结果是 nums[left]？
当 `left` 和 `right` 重合时，我们的搜寻范围已经缩小到了一个点。根据我们的二分规则，这个点必然是整段斜坡上的最低处。
