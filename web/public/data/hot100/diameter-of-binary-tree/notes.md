# 树中最远的两个小朋友：二叉树的直径

在二叉树这个大家庭里，每一个节点都是一个小朋友。有些小朋友住得很近，有些则住得很远。我们的任务是找到哪两个小朋友之间的距离最远。

**直径的奥秘**：
1. **最高峰的合影**：对于任何一个作为“中转站”的节点，经过它的最远路径，其实就是它的**左手臂长度**（左子树最大深度）加上它的**右手臂长度**（右子树最大深度）。
2. **全局大搜寻**：最远的路径不一定经过整棵树的根节点。所以，我们要巡视每一个节点，计算如果以它为“中转站”时，左拥右抱的长度是多少。
3. **金牌记录员**：我们准备一个记事本（全局变量），每算出一个节点的“左+右”长度，就和记事本上的最高纪录比一比，留下那个最大的。

虽然我们最后要的是“左+右”，但在巡视过程中，每个小朋友向上级汇报的还是自己这支队伍的“最大高度”（`max(左, 右) + 1`）。
