# 🧺 子集：我的魔法野餐篮

小朋友，想象你有一个篮子，里面有几个不同的水果，比如：苹果、香蕉、橘子。
你想带一些水果去野餐，你可以带：
*   **什么都不带**（空集 `[]`）
*   **只带一样**（比如 `[苹果]`）
*   **带两样**（比如 `[苹果, 香蕉]`）
*   **全带上**（`[苹果, 香蕉, 橘子]`）

所有这些可能的组合，就叫**子集**！

### 🪄 采摘步骤：

1.  **记录当前状态**：每当我们决定好篮子里放什么，就立刻在小本本上记下来。
2.  **按顺序挑选**：
    为了不重复，我们按顺序看水果。
    *   选了苹果后，我们只能在香蕉和橘子里选。
    *   选了香蕉后，只能在橘子里选。
3.  **回溯撤销**：
    试完了一种组合（比如“苹果+香蕉”），我们把香蕉拿出来，换成橘子试试（“苹果+橘子”）。

### 💡 核心秘诀：
每到一个新的水果面前，我们都面临一个选择：**“放进篮子”还是“不放进篮子”**。回溯法会帮我们遍历这所有的可能性！
