# 📚 括号生成基础百科

### 🌟 核心概念：有效括号
有效括号字符串满足两个条件：
1.  **左括号总数 = 右括号总数**。
2.  **任意前缀中，左括号的数量 >= 右括号的数量**。

---

### 📖 变量小字典：

*   **`symbol`**: 净剩余左括号数（左 - 右）。它是判断能否放右括号的关键。
*   **`index`**: 当前已填充的总长度。当 `index == 2n` 时，说明尝试结束。
*   **`cur`**: 我们的魔法画笔，记录当前的字符串序列。
*   **`al`**: 我们的结果仓库。

---

### 🔍 为什么不用 symbol > n？
因为我们只需要生成 $n$ 对括号，所以左括号最多只能放 $n$ 个。如果 `symbol` 加上已经闭合的右括号数量超过 $n$，那就不符合题目要求了。在代码里，我们直接控制左括号的总数不超过 $n$。
