# ⛺ 搭建最大的帐篷：柱状图里的秘密

小朋友，想象你有一排高低不平的柱子（柱状图）。你想在这些柱子之间搭一座**大帐篷**。
规则是：帐篷底必须贴着地面，而且帐篷的高度不能超过它覆盖的任何一根柱子。你的任务是：找出面积最大的那一座帐篷！

### 🪄 魔法实现：找找左右边界（单调栈）

对于每一根柱子，如果我们想让它成为帐篷的“屋顶”（即帐篷的高度就是这根柱子的高度），那这座帐篷能往左右两边伸展多远呢？
*   **向左看**：一直伸展到第一根**比它矮**的柱子为止。
*   **向右看**：一直伸展到第一根**比它矮**的柱子为止。

**如何快速找到这些“矮个子”邻居？**
我们用一个**“潜力观察区”（单调递增栈）**：
1.  **你是终结者吗？**：当你看到今天的柱子时，你看看观察区里最后进去的那根柱子。
    *   如果今天这根柱子**更矮**，那么对观察区里的那根高柱子来说，它的右边界找到了！
    *   **计算面积**：把高柱子拿出来。它的左边界就是现在观察区里剩下的那根柱子（因为剩下的都比它矮）。
    *   **算算看**：`宽度 = 右边界 - 左边界 - 1`，`面积 = 宽度 * 弹出柱子的高度`。
    *   **比比看**：看看这个面积是不是目前最大的。
2.  **加入观察区**：如果你比刚才的高，或者刚处理完前面的，你就进站排队，等你的右边界出现。

### 💡 为什么需要“哨兵”？
我们在数组最后加一个高度为 `-1` 的“假柱子”（哨兵）。这样可以保证所有的柱子最后都能被“逼”出来计算面积，不会有任何一根柱子被遗忘在观察区里。
