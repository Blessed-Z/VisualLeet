# 💡 查找范围巧记口诀

### 🌟 核心技巧：
> **“二分查找变一变，找到目标不间断；找头往左再逼近，找尾向右续前缘；记下索引当备胎，循环结束真身现。”**

### 🧠 记忆关键点：
1. **res 备胎变量**：初始设为 -1。每次 `nums[mid] == target` 时，先更新 `res = mid`，然后再继续移动边界。
2. **方向逻辑**：
   * `findFirst`: 找到相等时，`right = mid - 1`（向左逼近）。
   * `findLast`: 找到相等时，`left = mid + 1`（向右逼近）。
3. **循环条件**：使用标准的 `left <= right` 闭区间写法。
