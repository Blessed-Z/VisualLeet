# 📚 旋转数组搜索基础百科

### 🌟 核心概念：局部有序性
旋转数组其实是由两个升序数组拼接而成的。例如：`[4, 5, 6]` 和 `[0, 1, 2]`。
无论你在哪里切一刀，左边和右边里一定至少有一个是完全有序的升序数组。

---

### 📖 变量小字典：

*   **`left`**: 当前搜索范围的左边界。
*   **`right`**: 当前搜索范围的右边界。
*   **`mid`**: 探针位置。
*   **`target`**: 我们魂牵梦绕的目标。

---

### 🔍 为什么 `nums[mid] >= nums[left]`？
因为题目说数组原本是升序的。如果 `mid` 的值比 `left` 大或相等，说明从 `left` 到 `mid` 这一段还没有经历过旋转的“跳变”，所以它是单调递增的。
