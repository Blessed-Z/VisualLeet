# 🔢 找位次：魔法花园里的排队游戏

小朋友，想象你走进了一个**二叉搜索树（BST）**魔法花园。在这个花园里，所有的果实（节点）都严格遵守一个规则：**左边的果实总比树干小，右边的果实总比树干大。**

现在，我们要找出这个花园里**第 K 小**的那个果实。

### 🪄 找寻秘诀：

1.  **中序遍历（从小到大走一遍）**：
    如果我们按照“左边 -> 自己 -> 右边”的顺序去摘果实，你会神奇地发现，我们摘下来的果实正好是**从小到大**排列的！
2.  **设置计数器（小闹钟）**：
    我们带上一个计数器，每摘下一个果实，计数器就响一声。
3.  **抓准时机**：
    当计数器正好响到第 `K` 声时，嘿！手里的这个果实就是我们要找的宝贝啦！

### 💡 为什么 BST 的中序遍历是有序的？
这就是 BST 的神奇之处。因为它左边小、右边大，所以“先看左边、再看自己、最后看右边”就相当于从最小的开始，一步步走向最大的。
