# ☎️ 电话按键：解码神秘暗号

小朋友，你看过老式手机的按键吗？每个数字按键下面都藏着几个字母：
*   数字 `2` 下面有 `abc`
*   数字 `3` 下面有 `def`

如果有人给你发了一串数字暗号 `23`，它可能代表哪些字母组合呢？可能是 `ad`, `ae`, `af`, `bd`... 

### 🪄 解码步骤：

1.  **按下第一个键**：
    看数字串的第一个数字（比如 `2`），它有三个候选字母 `a, b, c`。
2.  **尝试一种可能**：
    假设我们选了 `a`。
3.  **按下第二个键（递归）**：
    去看下一个数字（比如 `3`），选它的一个字母（比如 `d`）。
4.  **拼成组合**：
    如果数字都按完了，我们就得到了一个暗号 `ad`。
5.  **换个姿势（回溯）**：
    把 `d` 拿掉，试试 `3` 下面的下一个字母 `e`，得到 `ae`。

### 💡 核心逻辑：
这就是一棵**多叉树**的遍历！每一个数字都是一层，每个字母都是一个分叉。我们从树根走到叶子，沿途就把所有的暗号都找出来啦！
