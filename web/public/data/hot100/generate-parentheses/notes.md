# ⛩️ 魔法门：对称的平衡艺术

小朋友，想象你是一个魔法建筑师。你要盖一排神奇的门，每扇门都有“左扇（`(`）”和“右扇（`)`）”。

盖门的规则非常严苛，否则魔法就会崩塌：
1.  **先开后关**：你不能先盖右扇门再盖左扇门。必须先有一个开启的动作，才能有闭合的动作。
2.  **总量守恒**：如果你要盖 $n$ 对门，那么左扇和右扇都必须正好用掉 $n$ 个。
3.  **不能落单**：在任何时候，右扇门的总数都不能超过左扇门的总数。

### 🪄 建筑步骤：

1.  **选择时刻**：
    每盖一个位置，你都有两个选择：
    *   **盖左扇门**：只要你手里还有剩下的左扇门（已用数量 $< n$），你就可以放心地盖。
    *   **盖右扇门**：这得小心！只有当目前**已经开启但还没关上**的门还有剩余时（左扇门数量 $>$ 右扇门数量），你才能盖右扇。
2.  **递归与回溯**：
    盖完一个，继续盖下一个。如果盖满了 $2n$ 个且正好全关上了，这就是一个完美的组合！记下来。
    然后，拆掉刚才那一扇，换另一种盖法试试。这就是回溯！

### 💡 核心秘诀：
这就像是**数左右手的差值**。左手代表“欠下的债”，右手代表“还债”。你不能还没借钱就先还钱（右比左多），最后还必须把债还清（左等于右）。
