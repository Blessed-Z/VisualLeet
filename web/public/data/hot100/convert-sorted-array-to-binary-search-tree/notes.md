# 108. 将有序数组转换为二叉搜索树 🌲

### 10 岁的小朋友，你好呀！今天我们来玩一个“排队变变变”的游戏！

想象一下，你有一排已经排好队的小士兵（有序数组），他们从矮到高整齐地站着。现在，我们要把他们变成一棵漂亮的二叉搜索树。

什么是二叉搜索树呢？就是中间的将军最高（根节点），左边的士兵都比他矮（左子树），右边的士兵都比他高（右子树）。

### 🌈 变身小秘诀：
如果我们想让树长得平衡（也就是左右两边一样重，不歪），我们应该选谁当将军呢？
没错！我们要选**最中间**那个小朋友当将军！

1.  **选中心**：找到队伍正中间的小朋友，让他坐到树的顶端。
2.  **分两边**：中心小朋友左边的人，全部去变成他的左边小分队。
3.  **再重复**：中心小朋友右边的人，全部去变成他的右边小分队。
4.  **一直做**：对每个小分队，我们都重复上面的动作，直到每个人都有了自己的位置。

### 🚀 奇妙的步骤：
1.  找到数组的中间位置 `mid`。
2.  把 `nums[mid]` 做成一个节点。
3.  用左边的部分去造左边的树。
4.  用右边的部分去造右边的树。
5.  最后，这棵完美的平衡树就造好啦！

这就像把一根细长的面条，从中间拎起来，它就自然地垂成了两边平衡的样子。
