# 📚 基础百科：大顶堆小词典

### 🌳 核心概念比喻：家族地位图

在我们的“堆”家族里，每一层都有规矩：
*   **大顶堆 (Max Heap)**：家长（父节点）的财富（数值）永远大于或等于他的两个孩子。
*   **完全二叉树 (Complete Binary Tree)**：家族里所有位置都是从左到右填满的，没有任何空隙。

---

### 🔍 变量小词典：
| 变量名 | 它的“外号” | 它是做什么的？ |
| :--- | :--- | :--- |
| `nums` | 小朋友队列 | 存储所有小朋友身高（数字）的数组。 |
| `k` | 第几个王 | 我们要找第几高的小朋友。 |
| `index` | 当前小官 | 在堆化过程中，正在被考察是否符合规矩的位置。 |
| `left` / `right` | 左右孩子 | 家族成员在数组里的邻居（`2*i+1` 和 `2*i+2`）。 |
| `max_index` | 最强勇士 | 在父子三人中，谁目前最强（值最大）。 |
| `end` | 竞技场边界 | 当前还在竞争的大王中，最后一个人的位置。 |

---

### 🧙 魔法公式：
*   **找儿子**：如果爸爸在 `i` 位置，他的大儿子在 `2*i+1`，二儿子在 `2*i+2`。
*   **找爸爸**：如果你在 `i` 位置，你爸爸在 `(i-1)//2`。
*   **第一个有儿子的爸爸**：他在全数组的 `(size // 2 - 1)` 位置！从他开始往回调整，整个家族就能建好堆啦！
