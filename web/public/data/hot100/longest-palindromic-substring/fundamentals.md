# 🧩 回文子串基础百科

### 🌟 核心概念：什么是“回文”？

回文（Palindrome）是指一个字符串正着读和反着读都完全一样。
*   **奇数对称**：中间有一个“中轴”字母，两边对称。如 `aba`, `radar`。
*   **偶数对称**：两边字母完全成对出现。如 `aa`, `abba`。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `s` | **纸带** | 需要处理的字符串。 |
| `left` | **左翅尖** | 向左扩散的当前指针。 |
| `right` | **右翅尖** | 向右扩散的当前指针。 |
| `expand` | **扩散魔法** | 一个小工具，用来不断向外检查两边字母是否相等。 |
| `res` | **最美记录** | 到目前为止，我们找到的最长的回文串。 |

---

### ⚠️ 敲黑板

*   **字符串切片**：Python 的 `s[i:j]` 会包含下标为 `i` 的字符，但**不包含**下标为 `j` 的字符。
*   **空串处理**：如果是空字符串或长度为 1 的字符串，它本身就是回文，直接返回。
*   **遍历次数**：总共要对 2n-1 个“中心”进行扩散检查。
