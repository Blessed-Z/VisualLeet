# 📚 拓扑排序与图基础百科

### 🌟 核心概念：有向图与环 (Directed Graph & Cycle)
*   **有向图**：城市与城市之间只有单行道。A -> B 说明可以从 A 到 B，但不一定能从 B 回到 A。
*   **环**：顺着单行道走，A -> B -> C -> A，走了一圈又回到了起点，这就是环。在选课问题里，环意味着死锁。

---

### 📖 变量小字典：

*   **`numCourses`**: 课程总数，相当于图里的节点（城市）数量。
*   **`prerequisites`**: 先修课规则表。
*   **`adjacency`**: 邻接表。记录了“学完这门课后，接下来可以解锁哪些课”。
*   **`flags`**: 状态记录本。记录每个节点是 0（没去过）、1（正在走）、还是 -1（已经证明是条活路）。

---

### 🔍 为什么叫“拓扑排序”？
把有向图里的节点排成一个线性序列，使得所有有向边都是从前指向后，这个过程就叫拓扑排序。如果图里有环，就永远排不出这个序列。我们这道题其实就是**利用 DFS 判环**。
