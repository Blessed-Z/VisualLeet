# 💡 思路巧记：零钱兑换诀

### 🎼 朗朗上口小口诀
> **“零钱兑换看总额，**
> **无限硬币随便拿。**
> **金额减去硬币值，**
> **加一比小不抓瞎！”**

---

### 🔑 核心关键点：
1. **完全背包问题**：硬币数量是无限的，这决定了我们可以从小到大顺序更新 `dp` 数组。
2. **初始化**：`dp[0] = 0` 是基础，其他位置初始化为 `amount + 1`（一个绝对达不到的最大值）。
3. **转移方程**：`dp[i] = min(dp[i], dp[i - coin] + 1)`。
    *   `i - coin` 代表用了这枚硬币后剩下的金额。
    *   `+ 1` 代表加上这一枚硬币。
4. **循环顺序**：外层遍历硬币（拿出一张硬币开始更新日记本），内层遍历金额（看这张硬币能帮我们优化哪些金额）。
5. **无解处理**：最后检查 `dp[amount]`，如果还是初始的最大值，说明拼不出来，返回 `-1`。
