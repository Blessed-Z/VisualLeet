# 📚 字符串解码基础百科

### 🌟 核心概念：嵌套结构处理
字符串解码本质上是在处理一种具有递归性质的嵌套结构。虽然可以用递归解决，但使用“栈”来模拟递归过程通常更直观、更高效。

---

### 📖 变量小字典：

*   **`num`**: 当前正在读取的倍数（可能有多位）。
*   **`res`**: 当前层级正在拼凑的字符串零件。
*   **`stack1`**: 存储之前层级的字符串。
*   **`stack2`**: 存储之前层级的倍数。

---

### 🔍 为什么 `res = cur_res + cur_num * res`？
`cur_res` 是在进入当前括号之前已经拼好的文字。
`cur_num * res` 是当前括号内重复后的文字。
把它们连起来，就得到了目前为止最完整的解码片段。
