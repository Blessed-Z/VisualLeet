# 💎 最小路径和：寻宝大冒险！

想象你在一个神秘的迷宫里寻宝。每个格子里都有一些**金币**，但这些金币很重。

你的目标是：从**左上角**走到**右下角**，带走的金币越少越好（因为你力气有限，想走得轻松点）。

规则还是老样子：只能**向右**或者**向下**走。

---

### 💡 魔法思路：最省力的选择

每当你走进一个格子时，你都要想一想：我是从哪儿过来的？

*   我可以从**左边**过来。
*   我也能从**上面**过来。
*   为了让手里的金币最少，我肯定会选那个**原来金币总数就比较少**的路径！
*   所以，到达当前格子的最少金币数 = **min(左边总和, 上面总和) + 当前格子的金币**。

---

### 🎨 寻宝步骤

1.  **起点**：`dp[0][0]` 就是第一格的金币数。
2.  **第一横排**：你只能从左边一直走过来，所以每格的总数就是前面的累加。
3.  **第一竖排**：你只能从上面一直掉下来，所以每格的总数也是上面的累加。
4.  **中间格子**：
    *   看看左边邻居，再看看上边邻居。
    *   谁家金币少，我就跟谁混！
    *   选个小的，加上当前格子的金币，记在心里。

最后右下角的数字，就是你走完整个迷宫能拿到的最少金币总数啦！🏆
