# 🧩 编辑距离基础百科

### 🌟 核心概念：什么是“编辑操作”？

编辑操作是文本处理中非常基础的概念。它定义了两个字符串之间的差异有多大。
这种差异也被称为 **Levenshtein 距离**。
它在拼写检查、DNA 序列比对、语音识别等领域都有巨大的作用。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `word1` | **旧模样** | 需要被整容的原单词。 |
| `word2` | **新模样** | 我们要达到的目标单词。 |
| `dp` | **手术报价单** | 二维数组，存放每个阶段的最小费用。 |
| `dp[i][j]` | **当前报价** | word1 的前 `i` 个变成 word2 的前 `j` 个最少需要多少步。 |
| `dp[i-1][j-1]` | **退一步的斜上** | 同时少看两个字母时的状态（对应替换）。 |
| `dp[i][j-1]` | **左边状态** | 目标多了一个字母（对应插入）。 |
| `dp[i-1][j]` | **上面状态** | 原单词多了一个字母（对应删除）。 |

---

### ⚠️ 敲黑板

*   **边界条件**：千万别漏掉 `i=0` 或 `j=0` 的初始值，它们是所有计算的基础。
*   **相同则不加**：如果字符一样，千万别手快去加 1，因为这次整容不用花钱。
*   **空间优化**：和 LCS 一样，由于只依赖于上一行和当前行的左侧，因此可以优化为 $O(n)$ 空间。
