# 💡 最长递增子序列巧记

### 🎵 押韵口诀
> 步步高升排排队，dp 数组记结尾。
> 初始长度全设一，双重循环找前序。
> 若是前人比我矮，我就排在他后边。
> 选个最长记下来，最后全局找第一。

---

### 🔑 核心关键点
1.  **定义状态**：`dp[i]` 是以 `nums[i]` 结尾的最长子序列长度。
2.  **转移方程**：`dp[i] = max(dp[j]) + 1`，其中 `0 <= j < i` 且 `nums[j] < nums[i]`。
3.  **结果**：答案是 `max(dp)`，不是 `dp[n-1]`。
4.  **进阶**：虽然 DP 是 $O(n^2)$，但有一种神奇的“贪心 + 二分”方法可以做到 $O(n \log n)$！
