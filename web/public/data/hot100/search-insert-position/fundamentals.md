# 📚 二分查找基础百科

### 🌟 核心概念：对数级别的时间复杂度 O(log N)
二分查找（Binary Search）是处理**有序数组**的高效算法。它的工作原理是不断将搜索区间减半。
如果你有 100 万个数字，二分查找最多只需要比较 20 次就能找到答案！

---

### 📖 变量小字典：

*   **`left`**: 搜索区间的左边界。
*   **`right`**: 搜索区间的右边界。
*   **`mid`**: 左右边界的正中间位置。计算公式：`left + (right - left) // 2`（这样写可以防止数字太大导致溢出哦）。
*   **`target`**: 你想要找的那个“新同学”。

---

### 🔍 为什么 `left = mid + 1`？
当我们确定 `nums[mid]` 比 `target` 小的时候，说明 `mid` 这个位置绝对不可能是我们要找的地方，所以我们要把左边界移到 `mid` 的下一个位置。
