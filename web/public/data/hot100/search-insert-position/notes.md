# 🔍 找位次：数字队伍里的插队高手

小朋友，想象你正在看一排小朋友按身高排队（从小到大）。这时候，又来了一个新小朋友 `target`，他想加入这个队伍，而且还要保持队伍依然是按身高排好的。

你的任务是：
1. 如果队伍里已经有跟他一样高的小朋友，就告诉他站在那个人的位置上。
2. 如果队伍里没有跟他一样高的，就帮他找一个位置插进去，让左边的都比他矮，右边的都比他高。

### 🪄 快速寻找法（二分查找）：

如果我们一个一个去比身高，那太慢了！我们要用**“对折查找法”**：

1. **确定范围**：伸出两只手，左手 `left` 指向队头，右手 `right` 指向队尾。
2. **看中间**：找到两手正中间的那个小朋友 `mid`。
3. **做判断**：
   * 如果 `mid` 比新同学矮，说明新同学应该往后站。我们把左手 `left` 往右移。
   * 如果 `mid` 比新同学高（或者一样高），说明新同学可能就在这里，或者应该往前站。我们把右手 `right` 移到 `mid` 的位置。
4. **收网**：当两只手碰头的时候，左手指向的位置就是新同学最完美的“落脚点”！

### 💡 为什么这个方法快？
每次我们看一眼中间的小朋友，就能排除掉**一半**的人！就像折纸一样，折几次就能从一大张纸里找到一个小点。
