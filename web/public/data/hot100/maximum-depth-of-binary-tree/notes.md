# 探险队的最高层级：二叉树的最大深度

想象你是一支探险队的队长，你的任务是测量一座神秘地下城堡（二叉树）到底有多少层。

测量方法非常简单但很有趣：
1. **派遣分队**：当你站在某个房间时，你会派出两个小分队，一个去左边的通道，一个去右边的通道。
2. **汇报高度**：每个分队回来后会告诉你，他们在那条通道里发现的最深层数。
3. **选出冠军**：作为队长，你会对比这两个数字，选出那个更大的（也就是更深的）。
4. **加上自己**：最后，别忘了加上你现在站的这一层！所以你的汇报结果就是 `max(左边深度, 右边深度) + 1`。

**特殊的出口**：如果某个通道走到底是个死胡同（节点为空），分队会汇报深度为 0。

这就是递归的神奇之处：每一层的房间都在做同样的事情，直到最后合力算出整座城堡的高度！
