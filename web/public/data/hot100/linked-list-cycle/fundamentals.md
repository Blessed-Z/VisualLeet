# 📖 基础百科：Floyd 判圈算法

## 💡 什么是 Floyd 判圈算法？
这就是我们用的“快慢指针法”。它不仅能判断有没有环，还能解决很多有趣的问题：
- 找环的入口（142题）。
- 找链表的中点。
- 甚至是判断一个数字是不是“快乐数”。

## 📋 变量小字典
- **`slow`**：我们的乌龟跑者。
- **`fast`**：我们的兔子跑者。
- **`fast.next.next`**：兔子的超能力，一次跨两个。

---

## 📈 进阶思考
为什么兔子每次跑 2 步一定能追上乌龟？跑 3 步行吗？
- 跑 2 步时，兔子和乌龟的相对距离每次缩小 1。距离缩小到 0 时就相遇了。
- 如果跑 3 步，距离每次缩小 2。如果初始距离是奇数，兔子可能会刚好“跨过”乌龟，需要多绕几圈才能追上。所以 2 步是最稳妥的！
