# 📚 基础百科：零钱兑换小词典

### 🌳 核心概念比喻：硬币实验室

在我们的“换钱之旅”中，记录仪会帮我们记下这件事：
*   **无限供应 (Infinite Supply)**：这是一种神奇的口袋，你想要多少枚 1 块钱、5 块钱，它都能拿出来。
*   **最省组合 (Optimality)**：我们的目标不是能凑出来就行，而是要用**最少**的数量。

---

### 🔍 变量小词典：
| 变量名 | 它的“外号” | 它是做什么的？ |
| :--- | :--- | :--- |
| `coins` | 硬币盒 | 装着不同面值的硬币（如 1, 2, 5）。 |
| `amount` | 目标总额 | 我们最后要凑齐的总钱数。 |
| `dp` | 省钱日记本 | 列表，记录每个金额对应的最少硬币数。 |
| `c` | 正在凑的钱 | 我们当前日记本记到了第几块钱。 |
| `coin` | 当前硬币 | 我们正在用哪种面值的硬币来试。 |

---

### 🧙 魔法公式：
*   **地基**：`dp[0] = 0`。
*   **更新法则**：`dp[c] = min(dp[c], dp[c - coin] + 1)`。
    *   `dp[c]`：原本在日记本上记的答案。
    *   `dp[c - coin]`：减去这枚硬币后，前面那个小金额的最佳答案。
    *   `+ 1`：因为我们现在多用了一枚 `coin` 硬币。
*   **判定**：如果最后 `dp[amount]` 还是很大，说明无解。
