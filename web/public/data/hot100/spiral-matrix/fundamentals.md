# 📖 基础百科：边界收缩法

## 💡 什么是边界收缩？
在处理二维空间（矩阵）问题时，如果遍历路径有明显的“层级”或“圈层”特征，使用边界变量是最直观的方法。
- 这种方法不需要复杂的数学公式，只需要模拟真实的物理过程。
- 它可以保证我们不重不漏地访问每一个元素。

## 📋 变量小字典
- **`up, down, left, right`**：四位“守门大将”，实时圈定当前的活动范围。
- **`range(right, left - 1, -1)`**：倒着走路的秘诀（逆序遍历）。
- **`while True`**：一直绕圈圈，直到碰到边界大门的禁令。
