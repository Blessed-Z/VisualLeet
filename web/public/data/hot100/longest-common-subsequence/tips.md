# 💡 最长公共子序列巧记

### 🎵 押韵口诀
> 两个字符串对对碰，二维表格记在胸。
> 字母相同左上找，加个一分最从容。
> 字母不同看左右，取个最大续前功。
> 全部扫完回头看，右下终点是真龙。

---

### 🔑 核心关键点
1.  **状态定义**：`dp[i][j]` 表示 `s1[0...i-1]` 和 `s2[0...j-1]` 的最长公共子序列长度。
2.  **状态转移**：
    *   `s1[i-1] == s2[j-1]`：`dp[i][j] = dp[i-1][j-1] + 1`
    *   `s1[i-1] != s2[j-1]`：`dp[i][j] = max(dp[i-1][j], dp[i][j-1])`
3.  **初始化**：通常表格的第一行和第一列都填 0（表示其中一个字符串为空时，公共长度为 0）。
4.  **常见变形**：可以求最长公共子串（要求连续），或者求最短公共超序列（包含两个序列的所有元素）。
