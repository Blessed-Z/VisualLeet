# 📖 基础百科：数学模拟

## 💡 什么是数学模拟？
在编程里，有些题目要求我们模拟真实的数学运算过程。
- 链表是逆序的，这反而帮了我们的忙！因为加法本身就是从个位（低位）开始算的。
- 如果题目给的是正序链表，我们可能还得先反转链表才能算。

## 📋 变量小字典
- **`carry`**：进位小助手（0 或 1）。
- **`total // 10`**：算出进位。
- **`total % 10`**：算出当前位留在新链表里的数。

## 📈 进阶思考
如果这道题要求返回的数字是正序的（比如 123 存为 1->2->3）怎么办？
- 那就是 LeetCode 的第 445 题啦。你可以利用“栈”或者“反转链表”的方法，先变回逆序，算完后再反转回去。
