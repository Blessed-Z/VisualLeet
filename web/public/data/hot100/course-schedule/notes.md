# 🎓 选课大挑战：打破无限循环

小朋友，想象你来到了魔法大学选课。
你想学“高级飞行课”（课程0），老师说：“必须先学会基础扫帚课（课程1）！”
你想学“基础扫帚课”，老师说：“必须先学会高级飞行课！”

糟糕！这变成了**死循环**，你永远也毕不了业了！这在图论里叫做**环（Cycle）**。

### 🪄 破解步骤（三色标记法）：

我们给每门课准备三个状态的牌子：
*   **0（白色/未探索）**：还没看过这门课。
*   **1（黄色/正在探索）**：我们正在看这门课的先修课链条，还没看到头。
*   **-1（绿色/安全）**：这门课和它的所有前置课我们都检查过了，绝对没有死循环，安全！

**检查每一门课：**
1.  **挂黄牌**：开始查这门课，先挂上黄牌。
2.  **顺藤摸瓜**：去查它的先修课。如果先修课是白牌，就按同样的步骤查它。
3.  **发现死循环！**：如果顺藤摸瓜时，突然发现某门先修课**正挂着黄牌**，说明我们绕了一圈又回到了原点，死循环出现了！直接报告失败。
4.  **安全通过**：如果所有先修课查完都没问题，我们就把这门课的黄牌摘下，换成**绿牌**。以后再查到这门课，看到绿牌就可以直接放心通过啦。
