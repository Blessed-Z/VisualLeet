# 💡 思路巧记：平方之王找寻诀

### 🎼 朗朗上口小口诀
> **“完全平方做积木，**
> **拼出 n 来最省数。**
> **减去平方再加一，**
> **最小值里找捷径！”**

---

### 🔑 核心关键点：
1. **状态定义**：`dp[i]` 是和为 `i` 的完全平方数的最少数量。
2. **转移方程**：`dp[i] = min(dp[i], dp[i - j*j] + 1)`。
    *   `j*j` 代表我们选择的一块“平方数积木”。
    *   `dp[i - j*j]` 是拼出剩余高度的最少数量。
    *   `+ 1` 代表加上当前这一块积木。
3. **初始化**：`dp[0] = 0`，其他位置初始化为正无穷 `inf`，这样才能在求 `min` 时更新。
4. **循环范围**：外层循环遍历数字 `1` 到 `n`；内层循环遍历平方数 `j*j`，直到 `j*j > i` 为止。
5. **预处理**：可以像用户代码那样，先把所有小于 `n` 的平方数列出来，提高内层循环效率。
