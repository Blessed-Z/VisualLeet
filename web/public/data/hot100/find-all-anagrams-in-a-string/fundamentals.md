# 📖 基础百科：collections.Counter

## 💡 什么是 Counter？
`Counter` 是 Python 为我们准备的**“智能计数器”**。
- 它就像一个能自动分类的储物箱。
- 你扔进去一个 "a"，它就记录 `a: 1`；再扔进去一个 "a"，它就更新为 `a: 2`。
- 最厉害的是，你可以直接用 `==` 来判断两个储物箱里的东西是否完全一样，这在算法里非常方便！

---

## 📋 变量小字典
- **`cnt_p`**：老师给的“标准配方清单”。
- **`cnt_s`**：我们餐盒里“现在的成分”。
- **`right`**：正在加入的食材位置。
- **`left`**：准备踢出的食材位置。
- **`enumerate`**：边跑边报数（同时给你索引和值）。

---

## 📈 进阶思考
虽然 `Counter` 很好用，但在某些极其追求性能的场合，手写 `[0] * 26` 的数组可能会稍微快那么一点点，因为省去了哈希表的开销。但在面试和实际开发中，`Counter` 的代码可读性是无敌的！
