# 镜像双胞胎：对称二叉树

想象一下，你面前站着一对双胞胎，他们正在做“镜像动作”。

要检查这棵树是不是对称的，我们就像在玩“找不同”：
1. **分身术**：我们假设有两棵一模一样的树（或者把根节点的左孩子和右孩子看作两棵独立的树）。
2. **同步检查**：我们派两个小侦探同时出发：
   - A 侦探看左边这棵树的**左手**，B 侦探就要看右边这棵树的**右手**。
   - A 侦探看左边这棵树的**右手**，B 侦探就要看右边这棵树的**左手**。
3. **心有灵犀**：如果两个侦探在每一个位置看到的数字都一模一样，或者同时走到了尽头（空节点），那这对双胞胎就是完美的“镜像对称”！

只要有一处不同，或者一个侦探看到了节点而另一个没看到，那他们就不是镜像双胞胎。
