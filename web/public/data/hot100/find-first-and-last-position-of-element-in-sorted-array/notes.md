# 📏 边界探测：找找数字双胞胎的头和尾

小朋友，想象你有一排穿着带有数字衣服的小朋友，他们按照数字从小到大排好队了。
有些数字可能会有很多个“双胞胎”甚至“多胞胎”聚在一起。

比如：`[5, 7, 7, 8, 8, 10]`，我们要找数字 `8` 的范围。
你会发现第一个 `8` 在位置 3，最后一个 `8` 在位置 4。所以范围就是 `[3, 4]`。

### 🪄 探测秘诀：

我们要用两次“对折查找法（二分查找）”来分别锁定**头**和**尾**。

1. **找头（寻找第一个出现的位置）**：
   * 当我们看到中间的小朋友正好穿着 `8` 号衣服时，我们不能停！
   * 因为左边可能还有更早出现的 `8`。所以我们要**继续往左边找**，把右手 `right` 移向 `mid - 1`。
   * 同时拿个小本本记下当前这个 `8` 的位置，防止左边再也找不到了。

2. **找尾（寻找最后一个出现的位置）**：
   * 同样，当我们看到中间的小朋友穿着 `8` 号衣服时，我们要**继续往右边找**。
   * 把左手 `left` 移向 `mid + 1`。
   * 也记下当前这个 `8` 的位置。

### 💡 为什么不直接往两边数？
如果数组里有 100 万个 `8`，一个一个数太累了！用二分查找，哪怕有再多的双胞胎，我们也只需要“对折”几次就能瞬间定位到最边上的那一个！
