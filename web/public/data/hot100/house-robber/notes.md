# 🦹 聪明的小怪兽：寻宝大作战

哈罗！小朋友，想象一下你是一个非常有礼貌的**小小寻宝怪兽**。你要走过一排房子，每间房子里都有一些金币。

### 👑 寻宝规矩
小怪兽有一个原则：**不能连续去两个相邻的房子**（因为如果连续去，警报就会叮铃铃响起来！）。我们要算出，小怪兽最多能带走多少金币？

---

### 🎒 魔法工具：决策笔记本（dp 数组）

我们要随身带一个**决策笔记本**：
*   **dp[i]**：记录当我们走到第 **i** 间房子时，能带走的最多的金币总数。

### 🎭 冒险步骤：

#### 第一步：打地基 📝
*   **第 0 间房**（还没开始）：金币是 0。
*   **第 1 间房**：只有一间房，当然是把这间的金币拿走啦！`dp[1] = nums[0]`。

#### 第二步：艰难的选择 🤔
现在小怪兽站在第 **i** 间房门口，他得做一个决定：
1. **选项 A（偷它！）**：如果拿了这一间的金币，那上一间房（第 i-1 间）就绝对不能去！所以总金币 = **前天（i-2）剩下的金币 + 今天的金币**。
2. **选项 B（跳过它）**：如果不拿这一间的，那小怪兽手里攒下的钱，就和走到**昨天（i-1）的时候一样多**。

#### 第三步：选最多的！ 🏆
小怪兽很聪明，他会比一比 A 和 B 哪个钱多，就选哪个！
公式：`dp[i] = max(昨天攒的钱, 前天攒的钱 + 今天的钱)`。

#### 第四步：算出终点 🏁
小怪兽每走一间房，就记一下当前能拿到的最高金额，直到走完最后一间！

---

### 💡 为什么这叫动态规划？
因为我们每走一步，都在根据之前的“最佳选择”来做现在的“最佳选择”。这就叫步步为营，最后就能拿到全场最高的奖金啦！
