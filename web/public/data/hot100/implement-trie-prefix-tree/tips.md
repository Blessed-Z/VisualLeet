# 💡 前缀树巧记口诀

### 🌟 核心技巧：
> **“字典套字典，字母当钥匙；一路往下找，没有就开辟；存完挂井号，查词看印记；前缀更简单，有路就胜利。”**

### 🧠 记忆关键点：
1.  **节点是什么**：在 Python 里，最优雅的实现是把每个节点当成一个普通的 `dict`（字典）。
2.  **结束标记**：一定要用一个特殊的字符（比如 `'#'`）来标记单词的结束。因为 `app` 和 `apple` 可以同时存在。
3.  **Search vs StartsWith**：它俩的代码几乎一模一样！唯一的区别是：`Search` 最后要检查 `return '#' in cur`，而 `StartsWith` 只要循环能跑完就直接 `return True`。
