# 💡 最长回文子串巧记

### 🎵 押韵口诀
> 回文串看蝴蝶膀，中心扩散最响亮。
> 奇数中心看单个，偶数中心看一双。
> 只要两边都一样，扩散开来变大强。
> 记住长度记位置，最后切下最长行。

---

### 🔑 核心关键点
1.  **两种中心**：遍历 `i` 时，要检查 `(i, i)`（奇数长度）和 `(i, i+1)`（偶数长度）。
2.  **扩散条件**：`left >= 0` 且 `right < len(s)` 且 `s[left] == s[right]`。
3.  **提取子串**：由于 Python 切片 `s[i:j]` 不包含 `j`，所以最终返回的应该是 `s[left+1 : right]`。
4.  **性能优势**：中心扩展法空间复杂度 $O(1)$，比 DP 的 $O(n^2)$ 空间更节省！
