# 🧩 不同路径基础百科

### 🌟 核心概念：什么是网格 DP？

网格 DP 就像是在玩棋盘游戏，你每一步的行动范围都受到了棋盘的限制。
在这个题里，你的动作只有“向右”和“向下”。
这意味着你要想到达某个格子，你的“前置条件”只有你的左边邻居和上边邻居。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `m` | **楼高** | 网格的行数（垂直方向）。 |
| `n` | **楼宽** | 网格的列数（水平方向）。 |
| `dp` | **路径统计图** | 一个二维数组，存放在每个格子时可能的路径总数。 |
| `dp[i][j]` | **当前格路径数** | 走到坐标 (i, j) 总共有多少种方法。 |
| `dp[i-1][j]` | **上边邻居** | 从上面一格走下来的方法数。 |
| `dp[i][j-1]` | **左边邻居** | 从左面一格走过来的方法数。 |

---

### ⚠️ 敲黑板

*   **1x1 的网格**：如果只有一格，路径数就是 1。
*   **第一行和第一列**：这些格子没有上面的邻居或者左边的邻居，所以路径数一直保持为 1。
*   **排列组合**：其实这道题也可以用数学公式（组合数）来解，因为你总共要走 (m-1) + (n-1) 步。
