# 💡 思路巧记：划分之王找寻诀

### 🎼 朗朗上口小口诀
> **“第一遍，记最后。**
> **第二遍，划边界。**
> **走到边界切一刀，**
> **段数越多越出色！”**

---

### 🔑 核心关键点：
1. **最后位置预统计**：先用一个哈希表或数组记录每个字母最后出现的位置。
2. **动态更新边界**：在遍历字符串时，通过 `max(end, last[c])` 不断更新当前片段能够到达的最远位置。
3. **分割时机**：当且仅当当前索引 `i` 等于当前片段的最远边界 `end` 时，说明当前片段已经包含了所有已出现字符的后续出现点，可以切割。
4. **贪心思想**：由于要在满足条件下划分为尽可能多的片段，所以一旦遇到可以切割的点（`i == end`）就立即切割。
5. **长度计算**：切割时的长度为 `end - start + 1`，切割后别忘了更新 `start = i + 1`。
