# ⚖️ 分割等和子集：天平平衡大挑战！

想象你有一堆金币，每个金币的重量都不一样（这就是 `nums` 数组）。

你的任务是：能不能把这堆金币分成**两份**，放在天平两端，让天平**完全平衡**？

---

### 💡 魔法思路：变身“凑数”游戏

1.  **算总重**：先算出所有金币的总重量 `total`。
    *   如果总重量是**奇数**（比如 11），那肯定分不平，直接认输。
    *   如果总重量是**偶数**，那我们的目标就是：能不能从金币堆里，挑出一些金币，它们的重量加起来刚好等于 **总重量的一半** (`target = total / 2`)？

2.  **魔法背包 (0/1 背包)**：
    *   这其实就是一个经典的“凑数”问题。
    *   我们有一个魔法口袋，它能记住目前为止我们能凑出来的所有重量。
    *   每拿出一个新金币 `x`，口袋里原有的每一个重量 `w`，都可以加上这个 `x`，变成一个新的重量 `w + x`。

---

### 🚀 终极武器：位运算魔法 (Bitset)

在 Python 中，我们可以用一个**超级长的二进制数**（就像一串超长的开关）来玩这个游戏：
*   这个二进制数的第 `k` 位如果是 `1`，代表重量 `k` 可以被凑出来。
*   刚开始只有第 0 位是 `1`（表示重量 0 肯定能凑出来）。
*   每遇到一个金币 `x`，我们就把整个二进制数**向左平移 `x` 位**，然后和原来的数做一次“或 (OR)”运算。
*   这相当于一次性把所有可能的状态都更新了！

最后看第 `target` 位是不是 `1` 就可以了。这就是所谓的“瞬移式”解法！⚡
