# 📚 基础百科

### 🌟 核心概念比喻

-   **平衡二叉搜索树 (AVL)**：想象一架精心设计的平衡秤，左边和右边挂的东西高度差不能超过 1 个台阶。
-   **二叉搜索树 (BST)**：一个有序的小岛，每个居民（节点）都得守规矩：左边的邻居必须比他小，右边的邻居必须比他大。
-   **分治法**：把一个大问题拆成几个小问题，分别解决后再拼在一起。比如把一个大队伍拆成两半，分别变树。

### 📖 变量小字典

-   `left`: 队伍的最左边（下标）。
-   `right`: 队伍的最右边（下标）。
-   `mid`: 队伍的正中间，用来当根节点的。
-   `nums`: 这一排排好队的士兵。
-   `root`: 我们正在建造的这棵树的最高指挥。
