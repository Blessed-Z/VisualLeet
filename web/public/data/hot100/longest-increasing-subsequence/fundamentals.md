# 🧩 LIS 基础百科

### 🌟 核心概念：什么是“子序列”？

子序列就像是从一串珍珠项链里挑出几颗。你可以跳着挑，但挑选的**左右位置不能调换**哦！
比如 `[1, 5, 2]`：
*   `[1, 5]` 是子序列。
*   `[1, 2]` 也是子序列。
*   `[5, 1]` 不是，因为原数组里 1 在 5 前面。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `nums` | **原队伍** | 输入的整数数组。 |
| `dp` | **最长小队表** | 记录以每个数为结尾的最长子序列长度。 |
| `dp[i]` | **小队长度** | 包含 `nums[i]` 在内的，最长的“步步高”小队人数。 |
| `i` | **当前小朋友** | 正在思考自己该排在哪里的小朋友。 |
| `j` | **前面的小朋友** | 尝试在他后面排队的小朋友。 |

---

### ⚠️ 敲黑板

*   **递增子序列**：每个数都必须比前一个数大。
*   **连续 vs 不连续**：子序列可以不连续，只要相对顺序不变。
*   **最大值**：最后的答案不一定是 `dp` 数组的最后一个位置，要把整个 `dp` 数组看完。
