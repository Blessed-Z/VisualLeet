# 🏝️ 岛屿探险队：寻找隐藏的大陆

小朋友，想象你是一名直升机飞行员，正在一片广阔的海洋（网格）上空巡逻。海洋里有些地方是海水（`0`），有些地方是陆地（`1`）。你的任务是数一数到底有几座独立的岛屿！

### 🪄 探险步骤：

1.  **全图扫描**：
    你开着直升机，从左上角开始，一排一排、一格一格地往下看。
2.  **发现新大陆！**：
    如果你看到了一块陆地（`1`），太棒了，这说明你发现了一座新岛屿！你在记事本上把**岛屿总数加 1**。
3.  **魔法沉岛（DFS 消除）**：
    为了防止下次巡逻时把同一座岛当成新岛，你派出了“魔法特种部队”降落在这个小格子上。
    他们有一种魔法，能把脚下的陆地变成海水（`grid[i][j] = '0'`）。然后，他们会继续向**上、下、左、右**四个方向进发。只要遇到相连的陆地，就继续用魔法变成海水。
    直到这座岛屿连着的所有陆地都“沉”入海底，特种部队才撤退。
4.  **继续巡逻**：
    直升机继续扫描。因为刚才那座岛已经被魔法变成了水，所以你再也不会重复计算它。下次再遇到 `1`，那绝对是另一座崭新的岛屿！

### 💡 为什么要把陆地变成水？
这就相当于给探索过的地方**打上记号**（插上小红旗）。如果不打记号，我们在它周围打转时就会无限死循环啦！
