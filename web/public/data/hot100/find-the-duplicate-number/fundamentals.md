# 🧩 寻找重复数基础百科

### 🌟 核心概念：什么是 Floyd 判圈算法？

弗洛伊德判圈算法（Floyd's Cycle-Finding Algorithm），也叫“龟兔赛跑算法”。
它通过两个步速不同的指针来检测链表或序列中是否存在环。
*   如果存在环，快慢指针一定会在环内相遇。
*   相遇后，将慢指针放回起点，快指针保持在相遇点，两人同速前进，再次相遇点即为环的入口。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `nums` | **迷宫地图** | 输入的包含重复数字的数组。 |
| `slow` | **乌龟** | 每次走一步的指针 (`slow = nums[slow]`)。 |
| `fast` | **兔子** | 每次走两步的指针 (`fast = nums[nums[fast]]`)。 |
| `nums[0]` | **起点** | 迷宫的入口。 |

---

### ⚠️ 敲黑板

*   **题目约束**：
    1. 不能更改原数组（假设数组是只读的）。
    2. 只能使用 $O(1)$ 的额外空间。
    3. 时间复杂度小于 $O(n^2)$。
*   **数学原理**：相遇点到环入口的距离，等于起点到环入口的距离（模环长度）。这就是为什么第二阶段同步走能碰到入口。
