# 📚 Trie (前缀树) 基础百科

### 🌟 核心概念：Trie
Trie 是一种专门处理字符串匹配的树形数据结构。它的特点是：**节点本身不存完整的单词，而是由节点之间的“边”（或者字典的键）来代表字母。** 从根节点走到某个节点，沿途经过的字母连起来，就是一个前缀或单词。

---

### 📖 变量小字典：

*   **`self.root`**: 树根。一个空字典 `{}`，所有单词的起点。
*   **`cur`**: 当前指针。就像你的手指，顺着字母一层一层往字典深处指。
*   **`'#'`**: 魔法印记。如果一个字典里有 `'#'` 这个键，说明走到这里可以拼成一个曾经存入过的完整单词。

---

### 🔍 嵌套字典的结构展示：
如果你存入了 `app` 和 `apple`，这棵树在 Python 里的长相其实是这样的：
`{'a': {'p': {'p': {'#': True, 'l': {'e': {'#': True}}}}}}`
