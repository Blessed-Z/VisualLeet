# 438. 找到字符串中所有字母异位词 (Counter 极简版)

## 💡 深度解析
小朋友，想象我们要制作一个**“字母三明治”**。
目标配方（字符串 `p`）已经写好了，我们要从一长串食材（字符串 `s`）里，找出所有符合配方的三明治。

### 核心策略：右进左出
我们用一个固定长度的**“滑动餐盒”**：
1. **右边加料 (right)**：每走一步，餐盒右边塞进一个新的食材。
2. **成分对比**：看看餐盒里的食材种类和数量，是不是跟目标配方一模一样。
3. **左边去料 (left)**：为了餐盒能继续往前滑，我们在移动前必须把最左边的那个食材扔掉。

这就是最简单的“进一个、比一下、出一个”的完美节奏！

---

## 🛠️ 执行步骤
1. **准备配方**：用 `Counter` 记下目标 `p` 的成分。
2. **开始滑行**：
   - 吞进右边的字母。
   - 算出左边的屁股在哪。
   - 如果盒子满了，就比一比。
   - 吐出左边的字母，腾出空间。
3. **记录成功**：只要配方对了，就把左边的起点记下来。

---

## 📈 复杂度分析
- **时间复杂度**：$O(N)$。我们只用了一个循环跑完整个纸带。
- **空间复杂度**：$O(K)$。`Counter` 最多也就装 26 种字母。
