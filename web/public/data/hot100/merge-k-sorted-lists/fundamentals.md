# 🧪 基础百科：合并 K 个升序链表

### 🌟 核心概念比喻

- **K 个链表**: 像是一组已经被整理好的小书架。
- **合并 (Merge)**: 像要把这些小书架上的书全部搬到一个超大书架上。
- **重新排序 (Reassembly)**: 把书全抱下架，然后在操场上按顺序排好，再一本本放到大书架上。

### 📖 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `lists` | 多组小队 | 输入的多个有序链表组成的列表 |
| `nodes` | 魔法口袋 | 存储从所有链表中提取出的数值 |
| `dummy` | 哨兵队长 | 指引新链表的头部，让第一个队员知道跟在谁后面 |
| `head` | 新队头 | 指向合并后的新超级链表的开始 |

### 🛠️ 工具箱

- **`list.sort()`**: 给数值列表排个队。
- **`ListNode(val)`**: 为每一个排好队的数字制作一个全新的“气球”（节点）。
- **`for...in` 循环**: 遍历每一队，并在这队里从小到大把数字取出来。
