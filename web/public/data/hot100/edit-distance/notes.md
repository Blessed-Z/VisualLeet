# ✂️ 编辑距离：单词整容手术！

想象你是一个“单词整容医生”。你的任务是把单词 A 变成单词 B。

你有三种手术工具：
1.  **插入 (Insert)**：在某个位置加一个新字母。
2.  **删除 (Delete)**：把某个位置的字母切掉。
3.  **替换 (Replace)**：把一个字母换成另一个字母。

每次手术都要收 1 块钱。我们要找的是：**最少花多少钱**就能完成整容？

---

### 💡 魔法思路：手术方案对比

我们用一张大表格来记录把 A 的前一段变成 B 的前一段需要多少钱。

当我们看 A 的第 `i` 个字母和 B 的第 `j` 个字母时：

1.  **字母一模一样**：
    *   太好了！这两个字母不用动。
    *   手术费 = **前面的手术费** (`dp[i-1][j-1]`)。

2.  **字母不一样**：
    *   我们需要动手术了，有三种方案：
        *   **替换**：把 A[i] 换成 B[j]。费用 = `dp[i-1][j-1] + 1`。
        *   **删除**：把 A[i] 删掉。费用 = `dp[i-1][j] + 1`。
        *   **插入**：在 A[i] 后面插一个 B[j]。费用 = `dp[i][j-1] + 1`。
    *   医生会选**最省钱**的那种方案！(`min(这三者)`)。

### 🎨 初始状态（没整之前的样子）

*   如果要把一个单词变成**空串**，唯一的办法就是把字母一个个全删掉（费用 = 单词长度）。
*   如果要把一个**空串**变成一个单词，唯一的办法就是把字母一个个全加上（费用 = 目标单词长度）。

这就是我们表格第一行和第一列的数字来源。

填满这张表，右下角的数字就是最省钱的手术方案总额啦！🏆
