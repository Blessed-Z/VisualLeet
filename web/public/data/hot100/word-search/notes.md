# 🔍 单词大搜索：地毯式寻宝

小朋友，想象你面前有一块巨大的拼字板（网格），上面乱七八糟地写满了字母。你的任务是：在这堆字母里，找出一条连续的路径，拼出指定的单词！

### 🪄 寻宝规则：

1.  **连连看**：你只能往当前格子的**上、下、左、右**四个相邻方向走。
2.  **不准回头**：在这个单词里，你不能两次使用同一个格子（不能原地打转）。
3.  **地毯式搜索**：由于不知道单词从哪儿开始，你得把每一个格子都当成起点试一遍！

### 🪄 探险步骤：

1.  **确定起点**：
    找找看，拼字板上哪个格子的字母跟你要找的单词的**第一个字母**一样？
2.  **深入探索（DFS）**：
    从这个起点出发，看看周围有没有单词的**第二个字母**。
3.  **隐身术（回溯的核心）**：
    当你走进一个格子时，为了防止自己待会儿绕回来踩到它，你先给它施个“隐身术”（比如暂时把它改为空白 `''`）。
4.  **分头行动**：
    你派四支小分队分别去四个方向找下一个字母。只要有一支队伍带回了好消息（找到了全词），你就大功告成！
5.  **现身（回溯还原）**：
    如果四支队伍都没找到剩下的字母，说明这条路走不通。在离开这个格子退回上一步之前，你必须解除“隐身术”，把格子**恢复原样**，好让别的搜寻路径以后还能用它。

### 💡 为什么叫“回溯”？
回溯就像是在笔记本上用铅笔写字。试错了没关系，拿橡皮擦掉（恢复原样），退回上一步重写就行啦！
