# 📖 基础百科：链表的“删减”

## 💡 链表怎么删除东西？
在链表里删除一个节点，就像是**“松手”**。
- 节点 A 原本拉着节点 B，节点 B 拉着节点 C。
- 如果我们要删掉 B：
  - 只需要让 A 松开拉着 B 的手，转而去拉住 C 的手。
  - 虽然 B 还在那，但它已经不属于这个队伍了（被“孤立”了）。

## 📋 变量小字典
- **`fast`**：那个负责拉开差距的“先行官”。
- **`slow`**：那个最终找到正确删除位置的“定位官”。
- **`slow.next.next`**：被删除节点后面的那个人。

---

## 📈 进阶思考
为什么一定要让 `slow` 停在目标的前一个位置？
- 因为你必须拿到“前驱节点”的手，才能改变指向。如果你直接停在目标节点上，你就没法控制前一个人拉谁了。
