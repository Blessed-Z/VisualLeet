# 📖 基础百科：矩阵旋转的数学规律

## 💡 旋转 90 度意味着什么？
在数学坐标系中，如果你把一个点 $(i, j)$ 顺时针旋转 90 度，它的新坐标会变成 $(j, n-1-i)$。
- **左上角** $(i, j)$ -> **右上角** $(j, n-1-i)$
- **右上角** $(j, n-1-i)$ -> **右下角** $(n-1-i, n-1-j)$
- **右下角** $(n-1-i, n-1-j)$ -> **左下角** $(n-1-j, i)$
- **左下角** $(n-1-j, i)$ -> **左上角** $(i, j)$

## 📋 变量小字典
- **`n`**：正方形图像的边长。
- **`i, j`**：当前选中的小拼图位置。
- **`n - i - 1`**：对应的镜像对称位置。
- **`matrix[i][j]`**：拼图上的一个具体颜色点。

## 📈 进阶思考
除了四角换位，还有别的原地旋转方法吗？
- 有！一个非常巧妙的办法是：
  1. 先按**对角线**翻转矩阵。
  2. 再按**左右垂直线**翻转矩阵。
- 两步翻转之后，矩阵也正好旋转了 90 度！这种方法虽然扫了两次，但逻辑更简单，不易出错。
