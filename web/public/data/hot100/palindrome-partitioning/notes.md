# ✂️ 切香肠：寻找完美的对称切片

小朋友，想象你手里有一根写满字母的长香肠。你的任务是：把它切成一段一段的，但是有一个非常奇怪的要求——**每一段切下来的香肠，上面的字母都必须是“回文”的！**

### 🪄 什么是回文香肠？
就是正着读和倒着读都一模一样的香肠。比如 `aba` 是回文，`level` 也是回文，但 `abc` 就不是啦。

### 🪄 探险步骤：

1.  **第一刀切在哪？**：
    从头开始，你可以试着在第一个位置切，或者第二个位置，或者更后面。
2.  **检查切片**：
    每切下一段，你都要拿起放大镜看看：**“这一段是回文吗？”**
    *   **是回文**：太棒了！把它放进你的“当前餐盘”（`path`）里，然后带着剩下的香肠，去重复这个切切切的过程。
    *   **不是回文**：哎呀，这一刀切坏了！赶紧把刀收回来，试着往后挪一点再切。
3.  **大功告成**：
    如果你刚好把整根香肠都切完了，而且每一段都符合要求，就把这整盘香肠的样子记在你的“菜谱大全”（`ans`）里。
4.  **回溯（撤销选择）**：
    记完一种切法后，为了尝试别的切法，你要把最后放进餐盘的那一段拿出来，重新放回剩下的香肠里，换个地方再切一刀试试。

### 💡 为什么叫“回溯”？
就像你在玩拼图。如果你发现这块拼图虽然能放进去，但会导致后面的拼不上了，你就要把它**拿出来**，退回上一步，试着放另一块。这就是回溯！
