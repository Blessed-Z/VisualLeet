# 💡 分割等和子集巧记

---

### 🔑 核心关键点
1.  **奇偶判断**：数组总和 `sum(nums)` 必须是偶数，目标值 `target = sum // 2`。
2.  **动态规划**：`dp[i]` 表示能否凑出重量 `i`。
3.  **内循环逆序**：如果使用一维数组，内循环必须**从大到小**遍历（避免重复选同一个数）。
4.  **位运算优化**：`bits |= (bits << x)` 是 Python 处理此类问题的绝杀技。
