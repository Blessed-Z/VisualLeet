# 💰 凑零钱：正好花光 target 块钱

小朋友，想象你兜里有几种面值的硬币（`candidates`），每种硬币你都有**无限个**。你的目标是：凑出正好 `target` 块钱！

### 🪄 凑钱规则：

1.  **大胆试错**：拿一个硬币，看看剩下的钱还需要多少。
2.  **无限续杯**：因为硬币无限，所以选了 2 块钱后，下次还可以继续选 2 块钱！
3.  **及时止损（剪枝）**：如果还没凑够，手里的钱就已经超过 `target` 了，那这条路就死掉了，赶紧回头（这就是**剪枝**）。
4.  **防止重复**：为了不搜出重复的组合（比如 `[2,3,3]` 和 `[3,2,3]`），我们约定：**只能选当前硬币或它之后的硬币**，不许走回头路！

### 💡 为什么先排序？
如果硬币按从小到大排好序，一旦我们发现“当前总和 + 这个硬币”已经超标了，那后面更大的硬币肯定也超标，我们就可以直接停止这轮搜索，省下好多力气！
