# 💡 思路巧记：双堆平衡诀

### 🎼 朗朗上口小口诀
> **“左大堆，右小堆，**
> **中位就在堆顶对。**
> **新数先去右边逛，**
> **左边接住再平衡。**
> **奇数取左顶，**
> **偶数取平均！”**

---

### 🔑 核心关键点：
1. **左侧大顶堆 (`self.left`)**：存储较小的一半元素，用负值来实现 Python 的小顶堆转大顶堆。
2. **右侧小顶堆 (`self.right`)**：存储较大的一半元素。
3. **数据流向**：
    *   为了确保 `left` 的最大值总是小于等于 `right` 的最小值：
    *   新数先压入 `right`，再将 `right` 的堆顶弹出并压入 `left`。
    *   如果 `left` 变得太臃肿（比 `right` 多了不止 1 个），就把 `left` 的堆顶调剂给 `right`。
4. **获取中位数**：
    *   奇数个：`left` 的堆顶（别忘了负号取反回来）。
    *   偶数个：`(left 堆顶 + right 堆顶) / 2`。
