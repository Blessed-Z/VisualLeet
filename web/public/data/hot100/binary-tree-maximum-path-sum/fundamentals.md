# 📚 最大路径和基础百科

### 🌟 核心概念：路径 (Path)
在二叉树里，路径就是你手指在树上划过的一条线，中间不能断开，也不能回头重复经过同一个点。

---

### 📖 变量小字典：

*   **`self.ans`**: 全局最大的彩虹桥总金额。
*   **`max_left`**: 左孩子能提供的最大正贡献。
*   **`max_right`**: 右孩子能提供的最大正贡献。
*   **`max_cur`**: 如果就在当前节点“转弯”，能达到的总值。

---

### 🔍 为什么 `max(..., 0)`？
如果一个子树的所有路径和加起来都是负数（比如 -5），那么加上它只会让当前路径的总金额变小。这时候我们宁愿“空手”，也就是取 0，代表不走那条路。
