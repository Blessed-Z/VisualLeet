# 💡 编辑距离巧记

### 🎵 押韵口诀
> 单词转换看最少，二维表格记手术。
> 字母相同照旧抄，斜上找数最划算。
> 字母不同看三项，左边上边和斜上。
> 选个小的加一分，整容费中最公道。

---

### 🔑 核心关键点
1.  **状态定义**：`dp[i][j]` 是把 `word1[0...i-1]` 变成 `word2[0...j-1]` 的最少步骤。
2.  **状态转移**：
    *   相同：`dp[i][j] = dp[i-1][j-1]`
    *   不同：`dp[i][j] = 1 + min(dp[i-1][j-1], dp[i][j-1], dp[i-1][j])`
3.  **三个动作对应**：
    *   `dp[i-1][j-1] + 1`: 替换 (Replace)
    *   `dp[i][j-1] + 1`: 插入 (Insert)
    *   `dp[i-1][j] + 1`: 删除 (Delete)
4.  **初始化**：第一行填 `0, 1, 2...`，第一列填 `0, 1, 2...`。
