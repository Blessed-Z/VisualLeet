# 📜 魔法卷轴：解开压缩的咒语

小朋友，想象你得到了一份神秘的魔法卷轴。卷轴上的文字被压缩了，看起来像这样：`3[a]2[bc]`。
它的意思是：`a` 要重复 3 次，`bc` 要重复 2 次。解码后的样子就是 `aaabcbc`。

卷轴还可以嵌套，比如 `2[3[a]b]`。这就像是套娃，得先解开最里面的，再解开外面的。

### 🪄 解码步骤：双口袋战术

我们带上两个魔法口袋：
1.  **倍数口袋 (`stack2`)**：专门存数字 $k$。
2.  **零件口袋 (`stack1`)**：专门存已经拼好的一部分文字。

**开始读卷轴：**
*   **遇到数字**：把它记下来（注意多位数，如 `12`）。
*   **遇到 `[`**：说明要进入一层新的“套娃”了。我们把刚才拼好的文字和倍数分别存进对应的口袋，然后清空手里的零件，重新开始。
*   **遇到 `]`**：说明最里面这层套娃拼好了！
    1. 从倍数口袋拿出倍数。
    2. 让手里的零件重复这么多遍。
    3. 从零件口袋拿出之前存的零件，把重复后的零件接在后面。
*   **遇到字母**：直接拿在手里，拼在当前零件后面。

### 💡 为什么需要两个栈？
因为我们要处理“套娃”结构。每遇到一个 `[` 就像是按下了“暂停键”，把当前进度存起来；每遇到一个 `]` 就像是按下了“播放键”，把刚才存的东西拿出来和现在的成果合并。
