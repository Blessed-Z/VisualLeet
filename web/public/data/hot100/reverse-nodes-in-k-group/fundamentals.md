# 📖 基础百科：链表的分段反转

## 💡 为什么这道题是 Hard？
因为这道题考察了链表最核心的三个难点：
1. **区间定位**：如何精准找到要反转的那一小段。
2. **局部反转**：如何在不破坏全局结构的情况下，让一小段节点调头。
3. **链条重连**：反转完之后，如何把这一段的“新头”接到前一段的“旧尾”上，把“新尾”接到下一段的“开头”上。

## 📋 变量小字典
- **`cur`**：当前组前面的那个节点（前驱）。
- **`tail`**：当前组的最后一个节点。
- **`tail.next`**：下一组的开头（后继）。
- **`index % k == 0`**：凑够一桌麻将（一组）的信号。

## 📈 进阶思考
这道题的递归版本怎么写？
- 递归思路：`找到第 k+1 个节点 -> 反转前 k 个 -> 把前 k 个的末尾连上对第 k+1 个节点递归调用的结果`。
