# 🍡 展开的基础百科

### 🧙 变量小字典 (The Magic Map)
| 变量名 | 它代表什么？ | 比喻 |
| :--- | :--- | :--- |
| `root` | 树的根节点 | 树的最顶端。 |
| `prev` | 上一个处理的节点 | 我们穿好的糖果串的第一个糖果。 |
| `node.left` | 左边的小果实 | 侧枝，我们要把它剪掉。 |
| `node.right` | 右边的小果实 | 下一根枝干，我们要用它来连接。 |

---

### 📖 核心概念：原地修改 (In-place)
原地修改的意思是：**你不能再买新糖果（创建新节点），只能把现有的糖果重新穿一穿。**

### 🔍 为什么用“反向前序”？
前序遍历是：根 -> 左 -> 右。
如果我们处理完“根”就把它的“左”接过来，那原来的“右”可能就会丢掉。
如果我们从最后面开始穿：右 -> 左 -> 根。
1. 当我们处理“根”的时候，“右”和“左”都已经准备好变成一串了。
2. 我们只需要把“根”放在这一串的最前面，整个过程就像搭积木一样稳固！
