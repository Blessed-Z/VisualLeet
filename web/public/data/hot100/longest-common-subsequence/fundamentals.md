# 🧩 LCS 基础百科

### 🌟 核心概念：什么是“子序列”？

子序列（Subsequence）是指从原序列中删掉一些元素后得到的新序列。
注意：子序列**不要求连续**，但**相对位置必须保持一致**。
比如 `[a, b, c, d, e]`：
*   `[a, c, e]` 是子序列。
*   `[a, b, d]` 是子序列。
*   `[e, a]` 不是子序列，因为原序列中 `a` 在 `e` 前面。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `text1` | **你的序列** | 第一个输入的字符串。 |
| `text2` | **他的序列** | 第二个输入的字符串。 |
| `dp` | **默契表** | 二维数组，存放各阶段的最长长度。 |
| `dp[i][j]` | **当前分数** | text1 的前 `i` 个字符和 text2 的前 `j` 个字符的默契分。 |
| `dp[i-1][j-1]` | **退一步的状态** | 同时去掉当前两个字母后的分数。 |
| `max(dp[i-1][j], dp[i][j-1])` | **二选一的最佳成绩** | 去掉其中一个后的最高分。 |

---

### ⚠️ 敲黑板

*   **空串处理**：如果任何一个字符串是空的，最长公共长度一定是 0。
*   **空间优化**：由于 `dp[i][j]` 仅依赖于上一行 `dp[i-1]` 和当前行的左侧，因此可以将空间复杂度优化到 $O(n)$。
*   **LCS vs LIS**：LCS 是处理两个序列，LIS（最长递增子序列）是处理一个序列。
