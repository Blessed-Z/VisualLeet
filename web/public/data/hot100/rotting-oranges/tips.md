# 💡 腐烂的橘子巧记口诀

### 🌟 核心技巧：
> **“僵尸好人分两边，按圈扩散算时间；新僵尸变老僵尸，好人清零大团圆；若有幸存躲天边，返回负一报不全。”**

### 🧠 记忆关键点：
1.  **集合操作超简单**：用 Python 的 `set`（集合）来记录 `rotten` 和 `fresh` 的坐标，找相邻橘子时直接用集合推导式，代码极其简洁！
2.  **边界条件**：如果一开始就没有新鲜橘子（`not fresh`），直接返回 0 分钟，别进循环。
3.  **提前终止**：如果在 `while fresh:` 循环里，发现这一轮没有产生任何新僵尸（`not rotten`），说明剩下的好人永远安全了，直接返回 -1。
