# 🧩 只出现一次的数字基础百科

### 🌟 核心概念：什么是异或 (XOR)？

异或是一种二进制运算，符号通常是 `^`。
*   **相同为 0**：`1 ^ 1 = 0`, `0 ^ 0 = 0`。
*   **不同为 1**：`1 ^ 0 = 1`, `0 ^ 1 = 1`。

在整数层面上，异或可以看作是**“无进位的二进制加法”**。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `nums` | **舞会人群** | 输入的整数数组。 |
| `ans` | **魔法棒/最终人选** | 累计异或后的结果，最后存储那个只出现一次的数。 |
| `x` | **当前点的名** | 正在遍历的数组元素。 |

---

### ⚠️ 敲黑板

*   **唯一性**：一定要确保其他数字都成对出现，异或大法才能奏效。
*   **初值 0**：因为任何数异或 0 都等于它自己，所以 `ans` 要从 0 开始。
