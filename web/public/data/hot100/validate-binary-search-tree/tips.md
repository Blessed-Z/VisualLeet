# 💡 思路巧记

### 🌟 记忆口诀
**验证二叉搜索树，**
**上下界限守得住。**
**左边更新最大值，**
**右边更新最小值。**
**根节点是无穷大，**
**每一关都不落下。**
**一旦发现出界了，**
**立马返回不真话！**

### 🔑 核心关键点
1.  **上下界递归**：这是最直观且准确的方法。每个节点在被访问时，都要通过父节点传下来的范围限制。
2.  **左子树范围**：`(min, node.val)`。
3.  **右子树范围**：`(node.val, max)`。
4.  **注意边界**：节点的值不能等于边界，必须严格小于或大于。
5.  **另一种思路**：中序遍历。如果是 BST，中序遍历出来的序列应该是**严格递增**的。
