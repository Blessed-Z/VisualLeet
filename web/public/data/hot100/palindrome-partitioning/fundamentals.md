# 📚 分割回文串基础百科

### 🌟 核心概念：回文 (Palindrome)
回文是指一个字符串正着读和反着读都一模一样。
*   数学定义：$s[i] = s[n-1-i]$ 对于所有 $i$ 成立。
*   Python 技巧：`s == s[::-1]`。

---

### 📖 变量小字典：

*   **`s`**: 原始的长香肠（待分割的字符串）。
*   **`path`**: 当前的餐盘，记录目前已经切好的回文片段。
*   **`ans`**: 菜谱大全，记录所有成功的分割方案。
*   **`i`**: 游标，代表当前还没切的那部分香肠是从哪个位置开始的。
*   **`j`**: 尝试切一刀的位置。

---

### 🔍 递归深度：
这道题的递归深度最大就是字符串的长度 $N$（即每一个字符都切一刀的情况，如 `"aaaa"`）。
