# 🍊 橘子感染记：僵尸橘子的扩散

小朋友，想象你有一个神奇的果篮（网格），里面装满了橘子。
有的橘子是**新鲜的**（`1`），有的是**僵尸橘子（腐烂的）**（`2`），还有的地方是**空的**（`0`）。

僵尸橘子有一种可怕的魔力：**每一分钟**，它们都会把上下左右紧挨着自己的新鲜橘子“咬”一口，让它们也变成僵尸！

你的任务是：计算需要**几分钟**，果篮里所有的橘子都会变成僵尸。如果有些橘子藏在角落，僵尸永远咬不到它，那就返回 `-1`。

### 🪄 扩散步骤：

1.  **清点阵营**：
    一开始，我们先找出所有的“初始僵尸”（放进一个集合里），并数清楚一共有多少个“新鲜橘子”。
2.  **僵尸出击（按分钟扩散）**：
    *   **第一分钟**：所有的初始僵尸同时向四周出击。如果碰到了新鲜橘子，就把它们变成新僵尸。
    *   **第二分钟**：刚才那些“新僵尸”接力，继续向它们的四周出击。
    *   ...就这样一圈一圈地扩散。
3.  **时间停止**：
    当没有新鲜橘子可以被咬，或者新鲜橘子已经全部死光时，游戏结束。
4.  **核对结果**：
    看看还有没有幸存的新鲜橘子？如果有，说明有些橘子是“绝缘”的（返回 `-1`）。如果没有，就报告一共花了多少分钟。

### 💡 为什么用 BFS 而不是 DFS？
因为腐烂是**同时向四周扩散**的，就像水波纹一样（BFS 广度优先搜索）。如果用 DFS（深度优先搜索），就像是一个僵尸一条路咬到底，这就无法正确计算“分钟数”啦！
