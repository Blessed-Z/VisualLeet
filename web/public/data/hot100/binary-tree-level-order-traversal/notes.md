# 坐电梯发糖果：二叉树的层序遍历

想象一下，你正提着一大袋糖果，来到一栋长得像二叉树的神奇大楼。你要给每一层的住户发糖果。

**发糖果的规则**：
1. **排队领糖**：大家都很文明，按照“先到先得”的原则排队。我们准备了一个大大的“魔法袋子”（队列）。
2. **第一层开始**：首先，根节点的老爷爷进袋子。
3. **一层一层来**：
   - 看看袋子里现在有多少人，这些人就是同一层的住户。
   - 把这一层的所有人依次请出来领糖果，并把领到的糖果记录在这一层的清单里。
   - 每个领完糖果的人，如果他有左孩子或者右孩子，就要赶紧把孩子们叫到袋子后面排队，等下一轮领糖果。
4. **电梯下降**：这一层发完后，我们就去领糖果清单里记下一笔，然后去发下一层。

直到袋子里空空如也，所有的住户就都领到糖果啦！
