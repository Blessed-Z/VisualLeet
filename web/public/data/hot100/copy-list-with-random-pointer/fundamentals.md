# 📚 复制链表基础百科

### 🌟 核心概念：深拷贝 (Deep Copy)
深拷贝就像你给好朋友写了一张贺卡，他觉得很好看，于是**自己也买了一张一模一样的纸，写了一模一样的字**。这就是深拷贝，如果他改了自己的贺卡，你的贺卡不会变。

---

### 📖 变量小字典：

*   **`head`** (首节点): 原项链的第一颗珍珠。
*   **`cur`** (当前指针): 就像我们的手指，在原链表上一个一个指着走。
*   **`dic`** (哈希表/魔法口袋): 专门用来记住“旧节点”和它对应的“新节点”之间的对应关系。
*   **`random`** (随机指针): 一根不听话的线，可能指向链表里的任何一个地方，甚至指向虚无（None）。

---

### 🔍 节点的结构：
1.  **`val`**: 节点里存的数字。
2.  **`next`**: 指向后面那颗节点。
3.  **`random`**: 随意指的一根线。
