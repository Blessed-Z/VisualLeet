# 🧩 颜色分类基础百科

### 🌟 核心概念：什么是荷兰国旗问题？

荷兰国旗（Dutch National Flag）由红、白、蓝三色条纹组成。
由计算机科学家 Edsger Dijkstra 提出，这个算法的目标是将数组中的元素划分为三个部分，分别对应三种颜色。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `nums` | **彩球筐** | 输入的待排序数组。 |
| `p0` | **红色边界** | 记录下一个 0 应该存放的位置。 |
| `p2` | **蓝色边界** | 记录下一个 2 应该存放的位置。 |
| `curr` | **巡逻员** | 当前正在检查的元素下标。 |

---

### ⚠️ 敲黑板

*   **交换逻辑**：
    *   `nums[curr] == 0`: `swap(nums[curr], nums[p0])`, `p0++`, `curr++`。
    *   `nums[curr] == 2`: `swap(nums[curr], nums[p2])`, `p2--`。
    *   `nums[curr] == 1`: `curr++`。
*   **空间复杂度**：$O(1)$，直接修改原数组。
*   **稳定性**：这种交换方法通常是不稳定的，但对于只有三种确定值的情况，稳定性不是主要考虑因素。
