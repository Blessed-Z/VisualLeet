# 🧩 多数元素基础百科

### 🌟 核心概念：什么是多数元素？

多数元素（Majority Element）在数学上定义为在一个长度为 $n$ 的数组中，出现次数严格大于 $\lfloor n/2 \rfloor$ 的元素。
注意：这意味着这样的元素**最多只能有一个**。

### 📚 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `nums` | **选民队伍** | 输入的整数数组。 |
| `ans` | **候选人 (Candidate)** | 当前在擂台上的数字。 |
| `cnt` | **候选人的血量 (Count)** | 记录当前候选人比其他人多出的票数。 |
| `x` | **当前选票** | 正在遍历的数组元素。 |

### ⚠️ 敲黑板

*   **投票抵消**：不同数字之间的抵消是算法的核心。
*   **最后检查**：在某些情况下（如果题目不保证一定存在多数元素），算法结束后还需要再遍历一遍数组，确认 `ans` 的出现次数是否真的超过了一半。但 LeetCode 的这道题通常保证一定存在。
