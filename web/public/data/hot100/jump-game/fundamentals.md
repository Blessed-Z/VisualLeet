# 📚 基础百科：跳跃小词典

### 🌳 核心概念比喻：超级望远镜

在我们的“跳跃之旅”中，记录仪会帮我们记下两件事：
*   **最远覆盖范围 (Max Reach)**：到目前为止，我们能够跳到的最远点。
*   **当前脚下砖块 (Current Position)**：我们现在走到了哪个位置。

---

### 🔍 变量小词典：
| 变量名 | 它的“外号” | 它是做什么的？ |
| :--- | :--- | :--- |
| `nums` | 砖块能量 | 记录每块砖头能给你提供多少跳跃力。 |
| `max_i` | 望远镜看最远 | 到目前为止，你最远能跳到哪块砖头上。 |
| `i` | 马里奥脚下 | 你当前走到了哪块砖头。 |
| `num` | 当前跳跃力 | 当前砖块能跳几格。 |

---

### 🧙 魔法公式：
*   **只要身在覆盖内**：`if max_i >= i:`（如果你能走到当前点）。
*   **尝试跳得更远**：`if i + num > max_i: max_i = i + num`（看看能不能刷新最远纪录）。
*   **最终答案**：就是走完全程后，看看 `max_i` 是否大于或等于最后一块砖头。
