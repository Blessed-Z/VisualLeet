# 📚 基础百科：高频统计小词典

### 🌳 核心概念比喻：快递分拣中心

在我们的“分拣中心”里，有两个重要的步骤：
*   **哈希表 (Hash Map)**：就像一张快递员的记账表。
    *   快递员数一数，1 号包裹有 3 个，2 号包裹有 5 个。
    *   记账表上就写着：`{1: 3, 2: 5}`。
*   **桶排序 (Bucket Sort)**：就像是快递架。
    *   架子上有很多层，每一层标着“出现次数”。
    *   3 次出现的快递统统放到第 3 层，5 次出现的快递统统放到第 5 层。

### 🔍 变量小词典：
| 变量名 | 它的“外号” | 它是做什么的？ |
| :--- | :--- | :--- |
| `nums` | 快递包裹 | 原始数组，里面有一大堆数字。 |
| `cnt` | 记账本 | 记录每个数字各出现了几次（`Counter` 对象）。 |
| `max_cnt` | 冠军票数 | 所有小朋友中最高的那个人得了多少票。 |
| `buckets` | 分类货架 | 每一层（索引）代表票数，层里的列表装的是拿到该票数的数字。 |
| `ans` | 颁奖名单 | 最后找出来的最频繁的 k 个数字。 |

---

### 🧙 魔法公式：
*   **第 `c` 个桶**：`buckets[c]` 里装的所有数字，它们的频率都是 `c`！
*   **从后往前看**：`for bucket in reversed(buckets)`。
    *   最后面那个非空桶里装的，肯定是整个公司最频繁的那些大王！
