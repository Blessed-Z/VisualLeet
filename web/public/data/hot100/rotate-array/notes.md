# 189. 轮转数组 (Rotate Array)

## 💡 深度解析
小朋友，想象有一列长长的**小火车**在铁轨上跑。
老师说：“把火车最后的 3 节车厢切下来，接到火车的最前面去！”

### 核心策略：切下来，粘上去
这其实是一个“剪切和粘贴”的过程：
1. **量长度**：如果要把 7 节车厢的火车向右转 8 次，那其实跟转 1 次是一样的（$8 \div 7$ 余 $1$）。所以我们先算算实际要搬几节车厢。
2. **切尾巴**：把火车最后面的 $k$ 节车厢整块切下来。
3. **接龙头**：把切下来的这一整块，啪嗒一下，粘到火车头的前面去。

这样，原来的尾巴就变成了新的火车头！

---

## 🛠️ 执行步骤
1. **取余数**：`k = k % len(nums)`。
2. **拿走尾巴**：找出最后 $k$ 个数。
3. **插入开头**：把这 $k$ 个数塞到索引 0 的位置。
4. **清理现场**：删掉原数组末尾多出来的重复部分。

---

## 📈 复杂度分析
- **时间复杂度**：$O(N)$。切片和插入的操作本质上还是要把元素挪一遍。
- **空间复杂度**：$O(N)$。在切片的过程中，Python 会产生一个临时的数组空间来存放搬运的内容。
