# 23. 合并 K 个升序链表：K 个小队大集合

想象一下，你面前有好多组小朋友（K 个小队），每组都已经按身高排好了队。现在，我们要把所有这些小队合并成一个**超级大队**，同样要按身高排好。

### 🏫 故事背景：大合影
为了让合影最快完成，我们想到了一个直观的方法：

1.  **点名出列**：把所有小队的小朋友全部叫出来，让他们先别管原来的队伍。
2.  **大操场集合**：让所有小朋友到操场中间乱哄哄地站在一起。
3.  **重新排队**：按照身高，重新给这群小朋友排队。
4.  **牵手成队**：排好后，让他们手拉手，组成一个长长的新链表。

### 🪄 魔法步骤

1.  **拆解 (Extraction)**：
    *   遍历每一个链表。
    *   把每个节点里的数字都拿出来，存进一个大列表（魔法口袋）里。
2.  **排序 (Sorting)**：
    *   给这个大列表里的所有数字念动“排序咒语”（使用 Python 自带的快速排序）。
3.  **重组 (Reassembly)**：
    *   根据排好序的数字，一个接一个地创建新的节点。
    *   把这些新节点串起来，就成了我们想要的超级大链表！

### 💡 为什么这么做？
虽然这看起来有点“暴力”，但它非常直观且易于实现。在数据量不是超级巨大的情况下，这是一种非常稳妥的方法。当然，如果你想更优雅一点，还可以使用“最小堆”或者“两两归并”的方法，那就像是更有序的合并过程啦！
