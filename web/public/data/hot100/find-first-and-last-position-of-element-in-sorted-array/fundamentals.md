# 📚 二分查找边界基础百科

### 🌟 核心概念：查找第一个和最后一个
在一个有序数组中，一个值可能会出现多次。普通的二分查找只要找到其中任意一个就会返回。
为了找到“最左侧”或“最右侧”的那个，我们需要在找到相等值后，**不停止搜索**，而是继续向对应的侧面挤压空间。

---

### 📖 变量小字典：

*   **`isFirst`**: 一个开关。`True` 代表我们在找头，`False` 代表我们在找尾。
*   **`res`**: 一个小笔记本，用来记录我们目前看到的“最像”边界的位置。
*   **`mid`**: 探路员，总是先去看中间的情况。

---

### 🔍 为什么结果是两个 [left, right]？
如果数组中根本没有这个数，我们的 `res` 就永远不会被更新，保持初始的 -1。所以最后返回 `[-1, -1]` 代表查无此人。
