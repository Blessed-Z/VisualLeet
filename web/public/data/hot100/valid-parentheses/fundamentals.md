# 📚 栈匹配基础百科

### 🌟 核心概念：栈 (Stack)
栈是一种“先进后出”（LIFO）的数据结构。想象一个自动取盘机，你放进去的第一个盘子在最下面，最后一个放进去的在最上面，而你只能取最上面的那个。

---

### 📖 变量小字典：

*   **`s`**: 我们要检查的括号字符串。
*   **`stack`**: 我们的魔法口袋（栈），只存左括号。
*   **`dic`**: 配对秘籍，记录了谁和谁是一对。
*   **`ch`**: 我们当前手里抓着的这只袜子（字符）。

---

### 🔍 为什么 `stack[-1]` 这么重要？
`stack[-1]` 代表栈顶元素，也就是口袋里最上面、最近放进去的那只左括号。只有它有资格和当前的右括号进行配对。
