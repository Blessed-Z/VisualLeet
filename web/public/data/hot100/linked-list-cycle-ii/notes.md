# 142. 环形链表 II (Finding the Entrance)

## 💡 深度解析
小朋友，还记得刚才的“龟兔赛跑”吗？
这次我们不仅要确认有环，还要找出那个**“环的入口”**。

### 核心策略：神奇的步数规律
1. **第一次相遇**：兔子追上乌龟后，我们确定了有环。
2. **重置跑者**：这时候，我们让一个跑者回到起点，另一个跑者原地不动。
3. **同速奔跑**：现在大家都保持“慢吞吞”的速度，一次走 1 步。
4. **揭晓谜底**：当他们再次撞个满怀时，那个相遇的位置，不多不少，正好就是环的入口！

为什么会这么神奇？这背后藏着一个精妙的数学小魔术，证明了起点到入口的距离，正好等于相遇点绕一圈回到入口的距离。

---

## 🛠️ 执行步骤
1. **判圈**：快慢指针走，直到相遇。
2. **重定位**：指针 A 留在原地，指针 B 飞回 `head`。
3. **寻找入口**：A 和 B 同时走，每次一步。
4. **终点**：相遇点即为入口节点。

---

## 📈 复杂度分析
- **时间复杂度**：$O(N)$。两阶段扫描，总步数依然和链表长度成正比。
- **空间复杂度**：$O(1)$。只用了几个标记位置的手指。
