# 128. 最长连续序列 (Longest Consecutive Sequence)

## 💡 深度解析
小朋友，想象有一大群数字在操场上乱跑。我们要找出**“排队最长的那一串数字”**。
比如有 `100, 4, 200, 1, 3, 2`。
虽然它们现在乱哄哄的，但其实 `1, 2, 3, 4` 是一串好朋友，它们的长度是 4。

### 核心策略：寻找“排头兵”
为了不重复数数，我们只从**每一串数字的“小队长”**开始数。
- 怎么判断谁是小队长？
- 只要看它前面的那个数（`num - 1`）在不在场就行了。
- 如果 `num - 1` 不在，说明它是第一个，也就是小队长！

---

## 🛠️ 执行步骤
1. 把所有数字扔进一个**魔法口袋**（Set），方便瞬间变出任何数字。
2. 拿起一个数字，问它：“你的前一名（`num - 1`）在口袋里吗？”
   - 如果在：它不是小队长，我们跳过它。
   - 如果不在：它是小队长！我们从它开始往后数 `num + 1`, `num + 2`...
3. 看看哪一队数出来的队员最多，记下最高分。

---

## 📈 复杂度分析
- **时间复杂度**：$O(N)$。虽然有两层循环，但每个数字最多只会被进入 `while` 循环一次。
- **空间复杂度**：$O(N)$。我们需要用口袋存下所有的数字。
