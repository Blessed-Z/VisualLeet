# 54. 螺旋矩阵 (Spiral Matrix)

## 💡 深度解析
小朋友，想象你在一个**方方正正的果园**里巡逻。
你要走遍果园里的每一个格子，但你有一个特别的习惯：你喜欢沿着最外面的墙走，走完一圈再往里面缩一圈，就像画蜗牛壳一样。

### 核心策略：贪吃蛇巡逻
我们要记住四个“边框”的位置：
1. **上边界 (up)**：最上面的那一排。
2. **下边界 (down)**：最下面的那一排。
3. **左边界 (left)**：最左边的那一竖。
4. **右边界 (right)**：最右边的那一竖。

你每走完一条边，这条边就算被你巡逻过了，可以往里“缩”一格。
就这样，你巡逻的地盘越来越小，直到所有的格子都被你踩过一遍！

---

## 🛠️ 执行步骤
1. **向右走**：走最上面那行（`left` 到 `right`），走完后 `up` 往下移。
2. **向下走**：走最右边那列（`up` 到 `down`），走完后 `right` 往左移。
3. **向左走**：走最下面那行（`right` 到 `left`），走完后 `down` 往上移。
4. **向上走**：走最左边那列（`down` 到 `up`），走完后 `left` 往右移。
5. **检查**：每走完一步，都要看看还有没有路可走。如果上下越界或左右越界，任务完成！

---

## 📈 复杂度分析
- **时间复杂度**：$O(M \times N)$。果园里每个果子（元素）你都只摘了一次。
- **空间复杂度**：$O(1)$。我们只记住了四个边界的位置，没占新地盘。
