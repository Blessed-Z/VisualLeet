# 🗺️ 寻宝图：找出总和达标的路径

小朋友，想象你在一棵大树上探险。树上的每个节点都藏着一些金币（也可能是欠债）。你的任务是：找出有多少段**连续的向下路径**，里面的金币加起来正好等于 `targetSum`。

### 🪄 魔法道具：
*   **前缀和地图 (cnt)**：这是一个神奇的账本。它记录了从“树根”到“当前位置”这一整条路上，所有出现过的累积金额。
*   **当前累积金额 (s)**：你从树根一路走过来，兜里揣着的总金币数。

### 🪄 寻宝步骤：

1.  **记录当前心情**：走到一个节点，把它的金币加进你的总数 `s` 里。
2.  **翻看账本**：如果你的总数是 `s`，你想找一段和为 `targetSum` 的路，那你只需要看看账本上有没有记录过 `s - targetSum` 这个数字。
    *   如果有，说明从那个记录点到你现在的位置，中间这一段正好就是我们要找的宝藏路径！
3.  **更新账本**：把你现在的总数 `s` 记入账本，告诉后面的队友。
4.  **继续探险**：去左边和右边继续找。
5.  **擦除足迹（回溯）**：当你离开这个节点回上一层时，要把刚才记下的 `s` 从账本里减掉 1。因为你的队友不能跨过你去别的分叉路找。

### 💡 为什么需要回溯？
因为路径必须是**向下**的。如果你从左边树林回到了树干，再去右边树林找，左边那些旧的记录就不能干扰右边的判断啦！
