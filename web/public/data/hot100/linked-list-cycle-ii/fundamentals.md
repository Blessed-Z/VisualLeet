# 📖 基础百科：相遇点与入口的距离

## 💡 那个神奇的数学证明
假设：
- 起点到入口距离为 $a$。
- 入口到相遇点距离为 $b$。
- 相遇点再走回到入口距离为 $c$。
- 环的长度为 $L = b + c$。

当兔子和乌龟第一次相遇时：
- 乌龟走了：$a + b$。
- 兔子走了：$a + b + nL$（$n$ 代表兔子多绕了多少圈）。
- 因为兔子速度是乌龟的 2 倍，所以：$2(a+b) = a+b+nL$。
- 简化一下得到：$a+b = nL$，也就是说 **$a = (n-1)L + c$**。
- 这说明：**从起点走 $a$ 步，正好等于从相遇点走 $(n-1)$ 圈再走 $c$ 步。** 结果都是回到入口！

---

## 📋 变量小字典
- **`ans`**：那个被派回起点的探路兵。
- **`slow`**：留在环内等待相遇的卫兵。
- **`return None`**：如果跑道是直的，直接告诉老师没环。
