# 📚 基础百科：爬楼梯小词典

### 🌳 核心概念比喻：马里奥的魔法步子

在我们的“爬楼之旅”中，记录仪会帮我们记下这件事：
*   **跳跃的方法 (Combinations)**：爬到每一层有多少种不同的方法。
*   **魔法记账本 (DP Table)**：一个装满数字的本子，记录每一层的答案。

---

### 🔍 变量小词典：
| 变量名 | 它的“外号” | 它是做什么的？ |
| :--- | :--- | :--- |
| `n` | 总层数 | 我们要爬到的最高目标。 |
| `dp` | 记账本 | 一个长度为 `n+1` 的列表，记录每一层楼梯的跳法。 |
| `dp[i]` | 第 i 层答案 | 爬到第 i 级台阶时，马里奥发现的所有方法数。 |

---

### 🧙 魔法公式：
*   **基础记忆**：`dp[0]=1, dp[1]=1`（没有台阶算 1 种，1 级也算 1 种）。
*   **神奇累加**：`dp[i] = dp[i-1] + dp[i-2]`（每一层的秘密，都藏在前两层的和里面）。
*   **最终答案**：就是走完全程后，看看 `dp[n]` 里记了多少。
