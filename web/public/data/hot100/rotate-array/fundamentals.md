# 📖 基础百科：Python 的切片赋值

## 💡 什么是切片赋值？
Python 的列表非常强大，你可以通过切片一次性替换掉列表里的一段区域。
- `nums[1:3] = [9, 9]`：把索引 1 到 2 的数换成两个 9。
- `nums[:0] = [5, 6]`：在最前面（索引 0 之前）插入 5 和 6。

## 📋 变量小字典
- **`size`**：火车的总长度。
- **`k % size`**：真正需要搬运的车厢数量。
- **`nums[-k:]`**：要搬运的那一截“尾巴车厢”。
- **`del`**：橡皮擦，擦掉不需要的部分。

## 📈 进阶思考
如果不允许使用额外的空间（$O(1)$ 空间复杂度）该怎么做？
- 有一个非常有名的“三次翻转法”：
  1. 全局翻转。
  2. 翻转前 k 个。
  3. 翻转剩下的。
- 这个方法就像“把整本书翻过来，再分别把两章翻回来”，非常精妙！
