# 📚 N 皇后基础百科

### 🌟 核心概念：国际象棋皇后攻击
皇后可以攻击同一行、同一列及两条斜线上的所有棋子。
*   **水平/垂直**：坐标中行相同或列相同。
*   **斜线 1 (左上到右下)**：在 $n \times n$ 网格中，同一条此类斜线上的点 $(r, c)$ 满足 $r - c$ 为常数。为了防止负数，通常用 $r - c + n$ 或直接用 $r - c$。
*   **斜线 2 (右上到左下)**：同一条此类斜线上的点 $(r, c)$ 满足 $r + c$ 为常数。

---

### 📖 变量小字典：

*   **`queens`**: 记录每一行皇后所在的列号。如 `queens[0] = 1` 表示第一行皇后在第二列。
*   **`col`**: 长度为 $n$ 的布尔数组，记录哪些列被占用了。
*   **`diaga1`**: 记录 $r+c$ 对应的斜线是否被占用。
*   **`diaga2`**: 记录 $r-c$ 对应的斜线是否被占用。

---

### 🔍 复杂度分析：
*   **时间复杂度 $O(N!)$**：虽然看起来像 $O(N^N)$，但由于每行只能放一个且有各种剪枝，实际复杂度接近 $N$ 的阶乘。
*   **空间复杂度 $O(N)$**：主要是递归深度和几个标记数组。
