# 15. 三数之和 (3Sum)

## 💡 深度解析
小朋友，想象我们要从一堆卡片里挑出三张，让它们的数字加起来正好等于 **0**。
这就像是在找“性格互补”的三剑客！

### 核心策略：排好队，定一个，搜两个
1. **排好队**：先让数字从小到大站好（排序）。这样我们就知道谁大谁小，找起来不迷路。
2. **定一个**：我们先选定一个“基础数”。
3. **搜两个**：剩下的两个数，我们用两个手指（双指针）分别从剩下的两头往中间指。
   - 如果三个人加起来太小了，左边的手指往右移（数字变大）。
   - 如果三个人加起来太大了，右边的手指往左移（数字变小）。

---

## 🛠️ 执行步骤
1. **排序**：让数组变成有序的。
2. **固定 i**：从头开始遍历，作为三剑客里的老大。
3. **设置 L 和 R**：
   - **L** 是老大的右边邻居。
   - **R** 是队伍最后一名。
4. **左右夹击**：
   - 如果 $nums[i] + nums[L] + nums[R] = 0$，记录答案！
   - 特别注意：遇到长得一样的数字要跳过，不能选重样的人。

---

## 📈 复杂度分析
- **时间复杂度**：$O(N^2)$。排序是 $O(N \log N)$，外层循环跑 $N$ 次，内层双指针跑 $N$ 次。
- **空间复杂度**：$O(1)$。我们就原地找，不需要额外的储藏室。
