# 🦒 最长递增子序列：排队比身高！

想象有一排小朋友在排队，每个小朋友都有自己的**身高**（这就是 `nums` 数组）。

我们要从这排人里挑出一些小朋友，组成一个新的**“步步高”小队**。规则有两个：
1.  **不能乱跑**：挑出来的小朋友，在原队伍里的先后顺序不能变。
2.  **必须更高**：新小队里，后一个小朋友必须比前一个小朋友**严格更高**。

我们要找的，就是这个“步步高”小队**最多能有多少人**？

---

### 💡 魔法思路：动态规划 (DP)

我们可以让每个小朋友都想一想：“如果我是这个‘步步高’小队的**最后一个人**，我的小队最长能有多少人？”

1.  **准备记录表 (dp 数组)**：
    *   `dp[i]` 表示以第 `i` 个小朋友结尾的最长小队长度。
    *   **初始状态**：每个小朋友自己都能组成一个小队，所以 `dp` 数组全是 `1`。

2.  **寻找偶像 (双层循环)**：
    *   小朋友 `i` 往前看，看他前面的所有小朋友 `j`。
    *   如果前面的小朋友 `j` 比自己矮 (`nums[j] < nums[i]`)：
        *   那小朋友 `i` 就可以排在 `j` 的后面！
        *   这时候小朋友 `i` 的小队长度就是 `j` 的小队长度加 1：`dp[j] + 1`。
    *   小朋友 `i` 会在所有能排进去的小队里，挑一个**最长的**记在自己的 `dp[i]` 里。

---

### 🎨 举个例子

队伍：`[10, 9, 2, 5, 3, 7]`

1.  **10**：自己一队，`dp[0]=1`。
2.  **9**：前面没比自己矮的，自己一队，`dp[1]=1`。
3.  **2**：同上，`dp[2]=1`。
4.  **5**：前面 **2** 比自己矮，`dp[3] = dp[2] + 1 = 2`。小队：`[2, 5]`。
5.  **3**：前面 **2** 比自己矮，`dp[4] = dp[2] + 1 = 2`。小队：`[2, 3]`。
6.  **7**：前面 **2, 5, 3** 都比自己矮。
    *   跟在 5 后面：`2+1=3`。
    *   跟在 3 后面：`2+1=3`。
    *   所以 `dp[5]=3`。小队可以是 `[2, 5, 7]` 或 `[2, 3, 7]`。

最后在记录表里找个最大的数，就是我们要的答案啦！
