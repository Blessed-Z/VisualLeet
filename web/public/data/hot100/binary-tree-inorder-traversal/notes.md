# 小蜗牛爬树：二叉树的中序遍历

想象有一只可爱的小蜗牛在二叉树上爬行。这棵树非常神奇，每一条树枝都是单向的。

中序遍历就像是小蜗牛的一种“左-中-右”探险：
1. **左转先锋**：小蜗牛总是先往左边的树枝爬，直到爬不动为止。它每经过一个节点，都会把这个节点先记在心里的“备忘录”（栈）上，因为它一会儿还要回来。
2. **打卡留念**：当左边没路了，小蜗牛会从“备忘录”里拿出最近的一个节点，吃掉那里的“能量豆”（记录节点值）。
3. **右转新世界**：吃完后，它会尝试往这个节点的右边树枝爬，去开启新一轮的探险。

为什么叫“中序”？因为对于任何一个小家庭（父节点和它的两个孩子），我们总是先去左边玩，回来吃掉爸爸给的糖果，再去右边玩。爸爸排在中间，所以叫中序！
