# 💡 排序链表：思路巧记

### 🎵 归并三部曲

> **快慢找中点，递归两边分，合并有序链。**

### 🔑 核心知识点

1.  **分治思想 (Divide & Conquer)**: 
    *   将复杂的大问题分解成类似的小问题（递归）。
2.  **快慢指针 (Slow & Fast Pointers)**: 
    *   `fast` 走两步，`slow` 走一步，当 `fast` 到底时，`slow` 就是中点。
3.  **合并技巧 (Merge Two Sorted Lists)**: 
    *   设置一个“虚拟头结点” (Dummy Node) 作为起始点。
    *   谁小就把谁接在后面。
4.  **递归出口**:
    *   当链表长度为 0 或 1 时，直接返回。
