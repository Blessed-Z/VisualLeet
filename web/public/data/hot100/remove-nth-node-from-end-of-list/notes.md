# 19. 删除倒数第 N 个节点 (The Distance Trick)

## 💡 深度解析
小朋友，想象有一排小朋友排队。老师说：“请让倒数第 2 个小朋友出列！”
可是你只能从前往后看，不知道谁才是倒数第 2 个。

### 核心策略：固定长度的小木棍
我们要怎么才能准确找到那个人呢？
1. **量好间距**：我们找来一根长度为 $n$ 的小木棍。
2. **定好两头**：让**兔子**拿住木棍的前头，**乌龟**拿住木棍的后头。
3. **一起走**：兔子和乌龟中间隔着这根木棍，大家一起往前走。
4. **到达终点**：当兔子的前头走出队伍时，乌龟的后头正好就指着倒数第 $n$ 个人的**前面那个人**！
5. **踢出队伍**：乌龟只需要松开原本拉着倒数第 $n$ 个小朋友的手，直接拉住更后面那个人的手，那个人就成功“隐身”出列啦！

---

## 🛠️ 执行步骤
1. 创建 `dummy` 节点指向 `head`。
2. 让 `fast` 指针先走 $n$ 步。
3. 让 `slow`（从 `dummy` 开始）和 `fast` 同步向后走。
4. 当 `fast` 到达末尾（None）时，`slow` 停在目标节点的前驱位置。
5. 执行 `slow.next = slow.next.next`。
6. 返回 `dummy.next`。

---

## 📈 复杂度分析
- **时间复杂度**：$O(N)$。我们只扫描了一遍链表。
- **空间复杂度**：$O(1)$。只用了两个标记位置的手指。
