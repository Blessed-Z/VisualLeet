# 📚 基础百科：杨辉三角小词典

### 🌳 核心概念比喻：数字传承

在我们的“数字金字塔”中，记录仪会帮我们记下这件事：
*   **层层相扣 (Hierarchy)**：每一层的数字，都是上一层数字的“后代”。
*   **首尾守卫 (Sentinels)**：每一行最左边和最右边的那个 1，就像守卫一样。

---

### 🔍 变量小词典：
| 变量名 | 它的“外号” | 它是做什么的？ |
| :--- | :--- | :--- |
| `numRows` | 总层数 | 我们要盖多少层的金字塔。 |
| `c` | 整个金字塔 | 一个二维列表，里面装着每一层的所有数字。 |
| `i` | 当前层数 | 正在搭建哪一层。 |
| `j` | 当前砖块 | 正在那一层的哪一个位置。 |

---

### 🧙 魔法公式：
*   **首尾自动填**：`c = [[1]*(i+1) for i in range(numRows)]`（预先把所有砖头都放好 1）。
*   **中间合体法**：`c[i][j] = c[i-1][j-1] + c[i-1][j]`（找到头顶上的一对爸爸妈妈）。
*   **层层构建**：从第 3 层开始算中间，因为前两层没中间位置。
