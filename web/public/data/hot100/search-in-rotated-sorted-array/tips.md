# 💡 搜索旋转数组巧记口诀

### 🌟 核心技巧：
> **“旋转数组不要慌，二分先看哪边强。一边单调必有序，另一边里藏玄机；目标若在有序处，缩减范围路自通；否则转战另一半，层层深入见真功。”**

### 🧠 记忆关键点：
1. **单调性判断**：`nums[mid] >= nums[left]` 判断左半段是否有序。
2. **范围检查**：在有序段内，用 `target >= nums[left] and target < nums[mid]` 判断是否在左侧。
3. **步长处理**：小心 `mid = left + (right - left) // 2` 这种标准写法，防止死循环。
