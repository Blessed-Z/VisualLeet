# 🍡 二叉树变糖果串：树到链表的变身术

### 🧒 小朋友讲故事
想象你有一棵漂亮的糖果树，有很多树枝和果实。现在，你想把这棵树变成一串**长长的、直直的糖果串**。
为了让味道最好，我们需要按照“前序遍历”的顺序来穿糖果：先穿根部，再穿左边的，最后穿右边的。

但是，如果你直接从头开始穿，你会发现左边的糖果还没穿完，右边的树枝就被你弄丢啦！

---

### 🎨 魔法步骤：倒着穿糖果
我们的魔法是：**先穿排在最后的糖果**！

1. **先穿最右边的**：因为它是链表的最后一部分。
2. **再穿左边的**：它是链表的中间部分。
3. **最后处理现在的根节点**：把它连到刚才穿好的“后半段”上，并把它的左边剪掉（因为它是直的，没有侧枝了）。
4. **不断更新“穿好的一段”**：让每个节点都知道它的下一个是谁。

---

### 💡 为什么这样做？
这就是“反向后序遍历” (右 -> 左 -> 根)。
当我们处理到一个节点时，它的所有“后面”的邻居都已经乖乖排好队了。我们只需要简单地把自己连到那排队的第一个人身上，变身就完成啦！
