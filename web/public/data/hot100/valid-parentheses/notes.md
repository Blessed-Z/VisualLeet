# 🧦 袜子配对：神奇的魔法口袋

小朋友，想象你有一堆乱七八糟的袜子，有圆形的（`()`）、方形的（`[]`）和花瓣形的（`{}`）。

有效的括号就像是在玩一个配对游戏，规则如下：
1.  **左边先出**：每当你看到一只左袜子（如 `(`），你就把它放进一个**魔法口袋**（栈）里。
2.  **右边配对**：当你看到一只右袜子（如 `)`）时，你必须立刻从口袋里拿出**最上面**的那只袜子看看。
    *   如果它们正好是一对（比如 `(` 和 `)`），那它们就开心地抵消了！
    *   如果对不上，或者口袋里根本没袜子了，那这个序列就是无效的。
3.  **最后清空**：等所有的袜子都看完了，如果魔法口袋里**一个也不剩**，说明大家都找到了自己的另一半，大功告成！

### 💡 为什么叫“栈”？
魔法口袋就是一个“后进先出”的栈。最后放进去的左袜子，必须最先出来和对应的右袜子配对。就像你堆积木一样，只有拿走最上面的，才能看到下面的。
