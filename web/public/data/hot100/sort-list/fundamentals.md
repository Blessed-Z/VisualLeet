# 🧪 基础百科：排序链表

### 🌟 核心概念比喻

- **链表 (Linked List)**: 像一串系在一起的气球。由于不是连续的盒子，我们不能直接跳到第 5 个气球，必须从第 1 个气球开始数。
- **虚拟头结点 (Dummy Node)**: 像是一根新的绳子头，方便我们把其他的气球往后系。

### 📖 变量小字典

| 变量名 | 比喻 | 作用 |
| :--- | :--- | :--- |
| `slow` | 蜗牛 | 慢慢走，用来定位中点 |
| `fast` | 兔子 | 跑得快，帮蜗牛找到终点 |
| `left` | 左边小队 | 左半部分排好序的链表 |
| `right` | 右边小队 | 右半部分排好序的链表 |
| `dummy` | 哨兵 | 辅助合并，保持新的链表整齐 |

### 🛠️ 工具箱

- **`mergeTwoLists(l1, l2)`**: 就像把两个整齐的书架合二为一，每次从两边各拿一本最薄的书，放在新的书架上。
- **`sortList(head)`**: 就像一个分队队长，不停地分人，直到分不动为止。
