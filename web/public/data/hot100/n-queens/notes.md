# 👑 皇后大作战：谁也不许打架！

小朋友，想象你有一群脾气非常大的“皇后”棋子。她们非常霸道，只要在**同一行、同一列、或者同一条斜线**上看到别的皇后，就会立刻吵起架来！

你的任务是：在 $n \times n$ 的大棋盘上，放下 $n$ 个皇后，让她们每一位都能开心地待着，谁也碰不到谁。

### 🪄 摆放秘诀：

1.  **逐行推进**：
    我们一排一排地放。既然每一行只能放一个皇后，那我们就从第一行开始找位置。
2.  **雷达探测（状态标记）**：
    每放一个皇后，我们就开启三个雷达：
    *   **垂直雷达 (`col`)**：盯着这一列。
    *   **左斜雷达 (`diaga1`)**：盯着从左上到右下的斜线。
    *   **右斜雷达 (`diaga2`)**：盯着从右上到左下的斜线。
3.  **试错与撤回（回溯）**：
    在每一行，我们试着在不同的列放皇后。
    *   如果雷达没报警，就放下皇后，去下一行继续放。
    *   如果放了之后发现后面的皇后没位置坐了，说明刚才那步放错了！我们就把刚才那个皇后**拿起来（回溯）**，关掉雷达，换一列再试。
4.  **记录队形**：
    当我们顺利放完 $n$ 行，所有人都不打架，这套队形就成功啦！记在小本本上。

### 💡 为什么斜线雷达那么神奇？
*   **左斜线**上的格子，它们的 `行号 + 列号` 是相等的！
*   **右斜线**上的格子，它们的 `行号 - 列号` 是相等的！
掌握了这个数学小秘密，我们就能秒速判断哪里危险啦！
