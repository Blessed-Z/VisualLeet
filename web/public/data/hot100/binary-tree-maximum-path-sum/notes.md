# 🌈 彩虹桥：寻找最值钱的穿树路径

小朋友，想象大树的每个节点都挂着一个不同面值的红包（有的红包里可能是欠条，也就是负数）。你要搭建一座**连续的彩虹桥**穿过这棵树，让桥上所有红包加起来的总金额最大！

### 🪄 搭建规则：
1.  **不能分叉**：桥必须是一条线，虽然它可以在某个节点“转弯”向下走，但不能同时向左、向右又向上。
2.  **有舍有得**：如果某个分支的红包全是欠债（负数），我们可以直接不要那个分支！

### 🪄 搭建步骤：

1.  **询问小朋友**：
    每个节点都会问它的左、右两个孩子：“你们能给我提供的最值钱的一条路是多少钱？”
2.  **计算“贡献值”**：
    孩子会回答一个数。如果孩子传回来的钱是负的，节点就说：“太穷了，我不要！”（即取 `max(child_sum, 0)`）。
3.  **尝试在这里转弯**：
    节点把自己、左孩子的贡献、右孩子的贡献全加起来：`node.val + left_max + right_max`。
    这就是**以当前节点为最高点（转弯点）**能搭出的最美彩虹桥。我们拿这个数去跟世界纪录（全局变量 `ans`）比一比。
4.  **汇报上级**：
    节点告诉自己的爸爸：“我能带给你的最值钱的一条路，是我自己加上（左路或右路中更值钱的那一条）。”

### 💡 为什么只能汇报一条路？
因为如果节点把左右两路都报给爸爸，爸爸再连接上自己，就会出现“分叉”，这就不是一条合法的桥啦！
