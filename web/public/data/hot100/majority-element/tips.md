# 💡 多数元素巧记

### 🎵 押韵口诀
> 多数元素过一半，同归于尽是绝招。
> 选个领头记候选，遇到相同加一票。
> 遇到不同减一票，票数归零换头号。
> 战场硝烟散尽后，最后留下是正道。

---

### 🔑 核心关键点
1.  **摩尔投票前提**：数组中必须存在出现次数超过 $n/2$ 的元素。
2.  **空间效率**：由于只用了两个变量（`candidate` 和 `count`），空间复杂度为 $O(1)$。
3.  **时间效率**：只需遍历一次数组，时间复杂度为 $O(n)$。
4.  **拓展**：如果要找出现次数超过 $n/3$ 的数，就需要准备两个领头羊和两个计数器。
