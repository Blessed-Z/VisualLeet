# 👨‍👩‍👧‍👦 寻根之旅：寻找最近公共祖先

小朋友，想象有两个好朋友 `p` 和 `q` 在森林大树里走丢了。你需要作为森林搜救员，找出他们向上数**第一个共同的集合点（祖先）**。

### 🪄 搜救策略：

1.  **如果你就是其中之一**：
    如果你走到了一个节点，发现它正好就是 `p` 或 `q`，那你赶紧举手大喊：“我在这儿！”并把这个消息传回上一层。
2.  **分头搜救**：
    在每个路口（节点），你派出一小队去左边森林找，另一小队去右边森林找。
3.  **汇总情报**：
    *   **两边都有喜报**：如果左边说找到了某人，右边也说找到了某人，那么**当前这个路口**就是他们汇合的最近地点！
    *   **只有一边有喜报**：如果只有左边找到了人（可能是找到了一个，也可能是两个都在那边汇合了），那就把左边的结果继续往上传。
    *   **都没找到**：那就告诉上级，这片林子没人。

### 💡 为什么这是最近的？
因为我们是从下往上汇报错报的。第一个收到“左右都有人”消息的节点，一定是离他们最近的那个家长。
