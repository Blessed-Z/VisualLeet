# 73. 矩阵置零 (Matrix Zeroes)

## 💡 深度解析
小朋友，想象有一栋大楼，里面有很多房间。有些房间里藏着**“小怪兽 0”**。
如果一个房间里有怪兽，那么这整层楼（行）和整竖排（列）的所有房间都要被封闭（变成 0）。

### 核心策略：在门口留便条
我们不想背着沉重的笔记本去记哪层楼要封，我们可以利用大楼的**第一层**和**第一列**作为“公告栏”：
1. **先记小秘密**：先看看第一层和第一列本来有没有怪兽，用两个小开关（`flag_row`, `flag_col`）记下来。
2. **去门口画叉**：去检查剩下的房间。如果发现 `(i, j)` 房间有怪兽，就跑到这一层的门口 `(i, 0)` 画个叉，再跑去这一列的楼顶 `(0, j)` 画个叉。
3. **大扫除**：看一眼门口和楼顶。只要门口有叉，或者楼顶有叉，这个房间就封掉！
4. **最后封门口**：最后根据最开始的两个小开关，决定要不要把“公告栏”（第一层/第一列）也封掉。

---

## 🛠️ 执行步骤
1. **状态记录**：记下首行、首列是否有 0。
2. **标记阶段**：遍历 `(1,1)` 到末尾，若 `matrix[i][j] == 0`，令 `matrix[i][0] = 0` 且 `matrix[0][j] = 0`。
3. **置零阶段**：再次遍历，若对应的首行或首列标记为 0，则该位置置 0。
4. **首部处理**：根据第 1 步的开关处理首行和首列。

---

## 📈 复杂度分析
- **时间复杂度**：$O(M \times N)$。矩阵被完整扫了两遍。
- **空间复杂度**：$O(1)$。我们直接在矩阵内部画叉，没多占地方。
