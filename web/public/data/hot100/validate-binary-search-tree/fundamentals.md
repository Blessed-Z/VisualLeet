# 📚 基础百科

### 🌟 核心概念比喻

-   **区间魔法**：就像玩“猜数字”游戏，爸爸说：“我猜 50，左边的小朋友肯定比 50 小，右边的小朋友肯定比 50 大。”
-   **二叉搜索树 (BST)**：一个非常有秩序的数字家族。对于任意一个节点，其左子树中的所有节点都比它小，右子树中的所有节点都比它大。
-   **中序遍历**：二叉树的一种行走方式。如果是 BST，中序遍历（左 -> 根 -> 右）会按从小到大的顺序排列。

### 📖 变量小字典

-   `node`: 我们正在检查的这个节点。
-   `lower`: 这个节点被允许的**最低限度**。
-   `upper`: 这个节点被允许的**最高限度**。
-   `float('-inf')`: 负无穷大，初始的最低限。
-   `float('inf')`: 正无穷大，初始的最高限。
-   `isValidBST`: 递归函数名，帮我们做检查的大法官。
