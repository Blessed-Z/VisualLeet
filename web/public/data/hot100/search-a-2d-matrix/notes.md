# 🏰 矩阵大冒险：城堡里的宝藏搜索

小朋友，想象你进入了一个整齐的城堡房间。房间里摆满了宝箱，横着看一排比一排大，竖着看一列也比一列大。

最特别的是：每一行的开头都比上一行的结尾还要大！这说明整个城堡其实是一条**弯弯曲曲的长蛇阵**。

### 🪄 寻宝秘诀：

我们要找一个特定的宝藏 `target`。如果一个一个开箱子太累了，我们要找一个聪明的**起点**。

1. **选择东北角（右上角）作为起点**：
   站在第一排的最右边位置 `(0, n-1)`。
2. **看宝箱里的数字**：
   * **正好是它**：哇！中大奖了，直接带走！
   * **太小了**：如果这个箱子比我们要找的还小，那这一排剩下的箱子（都在它的左边）肯定也太小了。我们不用看这一排了，**往下挪一行** (`i += 1`)。
   * **太大了**：如果这个箱子比我们要找的大，那它下面那一列的箱子肯定更大。我们不用看这一列了，**往左挪一列** (`j -= 1`)。
3. **循环往复**：
   就这样像走楼梯一样，要么下楼，要么往左走，直到找到宝藏或者走出城堡。

### 💡 为什么这个方法聪明？
每次你只要看一眼，就能排除掉**一整行**或者**一整列**！这比一个一个找要快得多啦！
