# 📖 基础百科：前缀积与后缀积

## 💡 什么是前缀积？
前缀积（Prefix Product）和前缀和很像：
- 它是指数组中从第 0 个到第 $i-1$ 个所有元素的乘积。
- 在本题中，我们通过一次循环就能动态地算出每个人对应的“左侧全家福”。

## 📋 变量小字典
- **`left`**：像一个背在左肩上的背包，越往右走，里面装的乘积就越重。
- **`right`**：像一个背在右肩上的背包，往回走时，把右边的乘积也装进来。
- **`dp`**：我们的最终礼物清单。

---

## 📈 进阶思考
如果数组里有一个 0 怎么办？有两个 0 怎么办？
- 如果有一个 0：除了 0 那个位置外，其他人的“超级礼物”都会变成 0（因为乘到了那个 0）。
- 如果有两个 0：所有人的“超级礼物”都会变成 0。
- 我们现在的这种“左右扫描法”天生就能完美处理 0 的情况，完全不需要特殊判断哦！
